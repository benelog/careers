## 이런 동료를 찾습니다.&#x20;

* JAVA + Spring 프레임워크에 익숙하신 분
* MSA / 글로벌 환경에서 분산처리와 이중화에 관심 있으신 분
* 대용량 / 대규모 트래픽 경험의 실무 경험이 있으신분&#x20;
* Docker 컨테이너 환경에서 실무 경험이 있으신분&#x20;
* github 를 이용한 개발, Unit Test와 CI/CD 실천에 익숙 하신 분
* Linux 환경의 인프라 운영 (Apache, NGINX, Tomcat)의 실무 경험이 있으신 분
