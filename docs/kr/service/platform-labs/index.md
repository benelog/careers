# 공통 플랫폼 기술 개발

Platform Labs는 네이버의 서비스에 쓰이는 공통 플랫폼을 개발하는 조직입니다. '네이버 서비스의 기초 체력을 튼튼하게 만들어주는 기술 개발'이 조직의 미션입니다.  서비스를 쉽고 빠르게 만들고 개선할 수 있는 안정적 플랫폼 기반을 제공하고 더불어 효율성과 개발 생산성을 높인다는 목표를 가지고 있습니다.

![](/assets/kr/image%20(3).png)



Platform Labs에서 개발한 공통 플랫폼 중에서는 오픈소스로 공개된 소프트웨어도 있습니다.&#x20;

* Pinpoint : [https://github.com/pinpoint-apm/pinpoint](https://github.com/pinpoint-apm/pinpoint)
* nGrinder : [https://github.com/naver/ngrinder](https://github.com/naver/ngrinder)
* NBase-ARC : [https://github.com/naver/nbase-arc](https://github.com/naver/nbase-arc)
* Billboard.js : [https://github.com/naver/billboard.js](https://github.com/naver/billboard.js)

Platform Labs에서 현재 채용을 진행 중인 직무에 대해서는 하위 페이지를 참고하시기 바랍니다.
