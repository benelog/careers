# AiRSearch

AiRSearch는 AI Recommender System 또는 AI Recommendation and Search의 약자로서 네이버의 추천과 검색의 개인화 추천/검색 플랫폼을 연구 개발하는 조직입니다. AiRSearch는 AI를 기반으로 뉴스, 블로그, 웹툰, 동영상 등 다양한 콘텐츠 추천서비스와 사용자가 입력한 질의와 취향에 따라 통합검색 랭킹을 개발하는 업무를 담당하고 있으며 아래 3가지 분야에서 열정 넘치는 여러분들의 지원을 기다리고 있습니다.
