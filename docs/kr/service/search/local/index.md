# 지역검색&추천

 Local AI는 네이버 지역검색과 위치기반 장소 추천 스마트어라운드의 서비스 및 기술을 연구 개발하는 조직입니다.

 지역검색은 '가로수길 맛집'과 같은 질의에 최적화된 검색결과를 제공하기 위해 질의 분석, 매칭, 랭킹 알고리즘을 연구하며, 스마트어라운드는 사용자의 위치, 연령, 성별, 시간과 같은 컨텍스트에 적합한 장소 추천을 연구합니다.

 이와 같은 검색과 추천은 NLP, Ranking, Recommendation, Embedding등 최신 AI기술을 기반으로하며 최대의 데이터를 활용하여 최고로 가치있는 서비스를 만들어 엔지니어를 찾고 있습니다!
