# Local Search

## 조직 소개 / 이런 일을 합니다.

네이버/LINE의 지역검색 서비스를 만들고 있습니다. 일상생활에서 사용자들이 가장 많이 사용하는 맛집/장소의 검색 고도화와 안정적인 서빙을 목표로 하고 있습니다. ML/모델링 담당자는 사용자의 방대한 데이터와 로그를 기반으로 다양한 AI 기술을 연구하여 이용자 검색 경험을 개선하는데 기여할 수 있고, Back-end 담당자는 빅데이터 기술과 각종 플랫폼을 활용하여 사용자를 이해하고 대규모 데이터를 처리하는 경험을 쌓을 수 있습니다.

## 역할 / 입사 후 담당 업무입니다.

* 지역검색 랭킹 고도화 및 피쳐 발굴을 위한 모델링
* BERT, GPT3 등의 최신 딥러닝 기술을 이용한 Topic 추출 및 검색의도 분석 연구
* Hadoop 및 Spark를 활용하여 검색 로그나 리뷰 문서에 대한 대용량 처리
* Cloud / Container 환경에서의 eco-system 및 open source 플랫폼을 활용한 분산처리 시스템 개발

## 자격 요건 / 업무를 위해 필요한 자격 요건입니다.

* 주도적으로 문제를 찾고, 해결할 수 있는 분
* Linux환경에서 개발 경험이 있으신 분
* C / C++ / Python / Scala / Java 중 1개 이상에 대한 활용 및 이해도가 높으신 분&#x20;
* 논리적인 사고와 체계적인 문제해결 능력이 있으신 분

## 우대 사항 / 추가로 보유하고 있다면 도움이 되는 사항입니다.

* Tensorflow/Pytorch 머신러닝 패키지를 사용하여 모델을 설계한 경험이 있는 분
* CNN, GANs, RNN/LSTM, BERT 관련 기술과 경험을 보유하신 분
* Hadoop / Spark 등 분산처리 플랫폼 개발 경험이 있으신 분
* 컨테이너 (Docker / Kubernetes) 환경에서의 개발 경험 및 NoSQL DB에 익숙하신 분
* Elasticsearch / Kafka / Spring / Hive 등의 오픈소스API를 활용한 개발 경험이 많으신 분
