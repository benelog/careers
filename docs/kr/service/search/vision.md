# Vision

Vision 팀의 미션은 우리 주변의 시각적 세계를 이해하는 머신러닝 알고리즘을 개발하고, Vision 기반의 새로운 사용자 경험을 네이버 검색 제품에 적용하는 것입니다.

**담당 업무**

* 네이버 이미지 검색
* 네이버 동영상 검색
* 네이버 스마트렌즈/쇼핑렌즈
* 이미지분류/객체인식/이미지리트리벌 등의 Vision 태스크 연구 개발
* 동영상 내용 기반 분석 관련 연구 개발
* 의미 기반 이미지-텍스트 매칭 모델 연구 개발&#x20;
* 머신러닝 기반 서비스 파이프라인 개발

**자격 요건**

* 컴퓨터과학, 컴퓨터엔지니어링, 수학 및 관련 분야 전공
* ML/DL에 분야에 대한 기본 지식
* PyTorch나 Tensorflow와 같은 딥러닝프레임워크에 대한 지식 및 개발 경험

**우대 사항**

* 머신러닝을 활용한 실무 경험
* Computer Vision 분야 top-tier 학회에 논문 게재 경험
* Kaggle 등 머신러닝 챌린지 입상 경험
* multi-gpu 및 distributed training 유경험자
