# Maps

네이버 지도 앱/웹 서비스와 지도 플랫폼을 맡고 있습니다. 네이버 지도에 지도 정보를 표현하기 위한 다양한 렌더링 엔진, 경로를 찾아주는 다양한 수단의 경로 탐색 엔진, 이동 안내를 담당하는 내비게이션 엔진, 검색 기능을 제공하기 위한 지도 검색 및 API 개발을 담당하고 있습니다. 사용자들의 생활을 연결해 주며 공간에 대한 정보를 잘 전달하기 위하여 기술을 연구하고 있습니다.
