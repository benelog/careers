# Back-End

### Atlas Platform

#### 이런 일을 맡고 있어요

* 지도 데이터에 대한 처리/가공/제공 역할을 담당합니다.
* 지도의 색상과 레이어의 우선순위등의 스일을 편집하고 제공하는 역할을 담당합니다.
* 공간 검색을 통해 데이터와 서비스 가치를 만들어 냅니다.

#### 입사하게 되면 담당할 업무입니다

* 심플 스타일 에디터 개발 및 공간 데이터 검색 API 개발

#### 기술 업무를 수행하기 위해 필요한 자격 요건입니다

* Java / Kotlin / React / Javascript / Go / NodeJS 중 1개 이상의 개발에 익숙하신 분
* 자기주도적인 업무태도와 동료와의 협업에 적극적이신 분
* 문제를 매우 깊게 파고들어서 새로운 문제를 찾고 해결하고 싶은 분
* 코드 리뷰 및 개발문화에 관심이 많은 분

#### 추가로 보유하고 있다면 좋을 스킬 셋이나 경험입니다

* Spring Framework을 이용한 경험이 풍부하신 분
* 대용량 트래픽에 대한 경험이 풍부하신 분
* 최적화 알고리즘 개발 및 성능 튜닝에 대한 경험이 풍부하신 분
* 트러블슈팅 경험이 풍부하신 분&#x20;

### Navigation Engine

#### 이런 일을 맡고 있어요

* 자동차,도보,자전거 경로 탐색엔진
* 안내 생성
* 도로 네트워크 데이터 처리

#### 입사하게 되면 담당할 업무입니다

* 자동차, 도보, 자전거 경로/안내 엔진 개발
* 엔진 서빙 API 스택 개발
* 내비게이션 주행 컨텍스트 데이터 관리
* 교통정보센터의 실시간 및 배치 데이터 파이프라인 개발 및 운영
* 정제 데이터 품질개선을 위한 통계적 모델 개발 및 평가
* 머신러닝 모델 서빙 API 스택 개발

#### 업무를 수행하기 위해 필요한 자격 요건입니다

* 기본적인 CS 기반 지식 (DS, Algorithm, OS, Database 등)
* Java / Scala / Python / C++ 중 1개 이상의 언어 사용 가능
* Kafka를 활용한 실시간 데이터 프로세싱 어플리케이션 개발 경험
* 웹어플리케이션 개발 및 서비스 경험

#### 추가로 보유하고 있다면 좋을 스킬 셋이나 경험입니다

* 최적화 알고리즘 개발 및 성능 튜닝에 대한 경험
* 어려운 문제 해결을 즐기시는 분
* 대용량 트래픽 처리 경험
* Spark 를 활용한 데이터 프로세싱 경험
* Airflow 및 Kubernetes 를 활용한 어플리케이션 운영 경험
* Spring framework 활용한 개발 경험
* 대용량 데이터 개발 경험&#x20;

### 대중교통

#### 이런 일을 맡고 있어요

* 대중교통 데이터 수급 및 처리
* 버스/지하철/시외/기차등의 API 서빙
* 버스/지하철 실시간 데이터 수급 및 제공
* 대중교통 최적 경로 탐색

#### 입사하게 되면 담당할 업무입니다

* 대중교통 서비스 API 개발
* 대중교통 데이터 파이프라인 구축
* 대중교통 데이터 관리 및 운영 페이지 개발
* 대중교통 데이터 기반 시간표 예측

#### 업무를 수행하기 위해 필요한 자격 요건입니다

* 데이터 파이프라인 설계 및 디버깅, 알고리즘에 대한 넓은 이해가 가능하신 분
* java / go 중 1개 이상의 개발에 익숙하신 분
* Spring Framework을 이용한 경험이 풍부하신 분
* 자기주도적인 업무태도와 동료와의 협업에 적극적이신 분
* 문제를 매우 깊게 파고들어서 새로운 문제를 찾고 해결하고 싶은 분
* 코드 리뷰 및 개발문화에 관심이 많은 분

#### 추가로 보유하고 있다면 좋을 스킬 셋이나 경험입니다

* 대중교통 데이터 경험이 있으신분
* 대용량 트래픽/실시간 데이터 프로세싱에 대한 경험이 풍부하신 분
* 최적화 알고리즘 개발 및 성능 튜닝에 대한 경험이 풍부하신 분
* 실시간 데이터를 활용한 서비스 개발에 관심이 많은 분&#x20;

### Service Platform

#### 이런 일을 맡고 있어요

* 지도서비 지도플랫폼을 위한 API 개발
* 지도서비스의 개발생산성을 높이기 위한 플랫폼 개발
* 지도검색(장소/주소/버스 검색 모델링, 컬렉션ETL, 엔진 및 API) 개발

#### 입사하게되면 담당할 업무입니다

* 알림 시스템 개발
* 지도앱/PC 홈 관련 서비스 api 개발
* 사용자별 검색이력/주소/장소 등의 데이터의 저장/처리/분석 등의 업무 개발
* 서비스에서 사용하는 플랫폼 개발(Metric, Monitoring, Logging, Gateway)

#### 기술 업무를 수행하기 위해 필요한 자격 요건입니다

* JAVA가 익숙하신 분
* Spring Framework 개발 능력이 있으신 분
* 기본적인 Linux 명령 사용 능력이 있으신 분
* RDBMS 사용 경험이 있으신 분

#### 추가로 보유하고 있다면 좋을 스킬 셋이나 경험입니다

* Spring Webflux 개발 경험이 있으신 분
* 이벤트 프로그래밍이 익숙하신 분
* 고가용성 및 대규모 트래픽 처리 설계 능력이 있으신 분
* Kubernetes 기반 서비스를 운영해보신 분
* NoSQL 사용 경험이 있으신 분
* Spring Cloud 개발 경험이 있으신 분
* Hadoop ecosystem 기반의 Spark 등을 이용한 대용량 데이터 경험이 있으신 분
* Kotlin, Scala, Spark, Lucene, Solr, Elastic Search 중 하나 이상의 사용 경험이 있으신 분
