# Data

### NAVER UGC 서비스

NAVER UGC는 [네이버 블로그](https://blog.naver.com), [네이버 지식iN](https://kin.naver.com), [네이버 인플루언서 검색](https://in.naver.com), [네이버 포스트](https://post.naver.com), [네이버 뿜](https://m.bboom.naver.com) 등 UGC 서비스를 개발하고 운영합니다. 매일 3,800만명의 사용자에게 하루에 3억 PV 이상 서비스 하고 있으며 1년 365일 무정지로 운영하고 있으며, 새로운 글로벌 UGC 서비스에도 도전하고 있습니다.

국내를 넘어 글로벌 최고의 UGC 서비스를 개발/운영하기 위해 항상 최고 수준의 UGC 관련 기술을 유지하고 있으며 지속적으로 기술을 업데이트 합니다.

### 이런일을 맡고 있어요

NAVER UGC 서비스의 Data TECH 을 담당하며, Data Science/Engineering으로 서비스 성장 및 신뢰성에 기여하는 것이 목표입니다.

#### Data Engineering

대량의 UGC 서비스 데이터의 수집과 편리한 데이터 인터페이스(Query, Workflow, Visualization)를 제공합니다. 데이터를 활용하는 서비스 기획/분석가들에게 사용 가이드와 강의를 통해 데이터의 가치를 높이고 서비스 개선의 기반을 제공합니다.

#### Data Science

UGC 서비스 데이터 분석을 통해 서비스를 개선하고 정책 결정에 기여합니다. 경험하기 힘든 대량의 정형/비정형의 UGC 데이터는, Data Scientist에게 분석의 즐거움과 성장의 기회가 될 수 있습니다.

### 입사하게 되면 담당할 업무입니다

#### Data Engineering

* UGC 서비스 로그 수집(ETL)&#x20;
* Spark 기반 데이터 파이프라인 운영
* 데이터 파이프라인 모니터링 대시보드 구축 및 이상탐지
* Data portal 웹, API 서버 개발/운영

#### Data Science

* UGC 서비스 정형/비정형 데이터 분석
* 비즈니스 지표 개발
* 이상탐지 및 서비스 이슈 분석/리포팅
* 콘텐츠/유저/상품 추천&#x20;
