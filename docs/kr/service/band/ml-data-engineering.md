# Business Data

## 커뮤니티 서비스 데이터 분석 및 광고 플랫폼 개발

### 입사하게 되면 담당할 업무입니다

* 빅데이터 기반으로 데이터 분석 도구 개발, 광고 서비스 및 최적화 등의 업무를 합니다.
* 데이터 지표를 기반으로 서비스를 개선하고 공동의 목표 달성을 추구합니다.
* 최신 기술 스택을 기반으로 다양한 기술 경험을 할 수 있습니다.
* Hadoop, Hive, Spark, Kafka, Flume, Elasticsearch, TensorFlow, SK-Learn
* 사용자 행동 분석, 비즈니스 및 마케팅 분석, A/B Test, 광고 플랫폼 개발, 광고 최적화, Anomaly detection

### 업무를 수행하기 위해 필요한 자격 요건입니다

* 논리적인 사고와 알고리즘에 능숙하신 분
* 객체 지향 개발, 디자인패턴, 대용량 데이터 설계에 대한 기본 지식을 가지신 분
* Java, Javascript 개발 경험과 프로그래밍 기본 지식을 가지신 분
* API 서버, 웹서버 개발과 유지보수, DB 모델링 및 시스템 설계 경험이 있으신 분

### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다

* 고성능/대용량 서비스 개발, 대용량 데이터 경험이 있으신 분
* Hadoop, Spark, Kafka 등 대용량 데이터 파이프라인 기술 경험이 있으신 분
* 데이터분석 및 통계 지식을 갖추신 분, 머신러닝 개발 경험자
