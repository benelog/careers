# Naver Game Server Dev

## 입사하게 되면 담당할 업무입니다

* 네이버 게임 커뮤니티 서비스 엔지니어링을 통해 사용성/서비스 개선을 합니다.
* e-sports 관련 기능을 설계하고 개발
* 고품질 라이브를 위한 기술과 방송도구 개발
* REST API 기반으로 다양한 app/web으로 서비스를 제공하고 정보 수집합니다.
* 대용량 트래픽 처리와 MSA 기반의 시스템 설계를 합니다.

## 업무를 수행하기 위해 필요한 자격 요건입니다

* 백엔드 개발 경력 2년 이상
* Spring Framework 이해 및 활용
* 웹 프로토콜 및 인프라에 대한 이해
* MSA 아키텍처 이해 및 RESTful API 설계
* GIT 사용 경험 및 이해

## 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다.

* 라이브 (스트리밍) 서비스 개발 경험
* 대용량 데이터 처리 개발 경험 우대
* 커뮤니티 서비스 개발 경험 우대
* Docker, Kubernetes등 컨테이너 플랫폼에서의 빌드 및 운영 경험
* 고성능/고가용성을 위해 고민하고 구현한 경험 우대

## 이러한 분과 함께 하고 싶습니다.

* 본질적인 원인을 파악하여 문제를 해결하시는 분
* 항상 성능에 대해 고민하며 동작 원리에 대해 궁금해하는 분
* 주도적이고 책임감 있게 업무를 처리하는 분
* 새로운 도전을 즐기며, 새로운 기술에 관심이 많으신 분
* 동료를 배려하며, 함께 성장하고자 하시는 분
