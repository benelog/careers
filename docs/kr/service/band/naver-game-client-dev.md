# Naver Game Client Dev

## 입사하게 되면 담당할 업무입니다.

* 네이버 게임 커뮤니티 서비스를 개발/개선 합니다.
* 네이버 e-sports 관련 기능을 설계하고 개발합니다.
* 고품질 라이브를 위한 기술과 방송도구를 개발합니다.
* 네이버 게임 App 신규 기능 개발, 기존 기능 개선 등의 업무를 합니다.
* 네이버 게임 SDK 를 개발/개선 합니다.

## 업무를 수행하기 위해 필요한 자격 요건입니다

### Naver Game Client Front-End&#x20;

#### 기본 요건

* 프론트엔드 개발 경력 2년 이상
* HTML, CSS, 최신 ECMAScript 에 대한 이해가 깊으신 분
* React, TypeScript, Node.js 등을 이용한 웹서비스 개발 경험이 있으신 분

#### 우대 요건

* 라이브 (스트리밍) 서비스 개발 경험
* Node.js 개발 및 운영 경험
* 반응형 웹 개발 경험

### Naver Game Client Android&#x20;

#### 기본 요건

* Kotlin 기반 Android앱 개발 경험(2년이상)
* Android 기본 Component를 이해하고있는 분(Activity, Service, BroadcastReceiver, ContentProvider)
* Android SDK UI Component로 자유롭게 UI를 구성할 수 있는 분
* Android Jetpack Compose 개발 경험이 있는 분 또는 기본 지식이 있는 분

#### 우대 요건

* Android Jetpack Compose로 서비스 개발 경험자 우대
* 라이브스트리밍(MediaPlayer, ExoPlayer) 서비스 개발 경험자 우대

### Naver Game Client iOS&#x20;

#### 기본 요건

* Swift 기반 iOS앱 개발 경험(2년이상)
* iOS 기본 지식과 Cocoa Touch 이해도가 높으신 분

#### 우대 요건

* RxSwift 개발 경험이 있으신 분
* MVP / MVVM / Clean Architecture 등 아키텍쳐 설계와 디자인 패턴을 적용한 개발 경험이 있으신 분
* 앱 성능 최적화, 로직 공통화 및 구조 개선에 경험이 많으신 분

## 이러한 분과 함께 하고 싶습니다.

* 품질 좋은 소프트웨어 개발을 위해 지속적으로 노력해오신 분
* 새로운 도전을 즐기며, 새로운 기술에 관심이 많으신 분
* 자신의 성장과 서비스의 성장을 함께 하고 싶으신 분
* 자기주도적인 업무태도와 동료와의 협업에 적극적이신 분
