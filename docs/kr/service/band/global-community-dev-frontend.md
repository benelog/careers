# Global Community Dev FrontEnd

## 입사하게 되면 담당할 업무입니다.

* 네이버의 글로벌 커뮤니티서비스들에 대한 웹 FrontEnd 개선 및 신규 개발을 담당합니다.
  * 이미 국내에 잘 알려진 BAND 서비스의 글로벌 최적화 및 개선 진행중
  * 새로운 글로벌 커뮤니티 서비스에 대한 도전중
* 밴드와 역사를 함께한 만큼 과거에서 최신까지의 웹 FrontEnd 환경을 경험 할 수 있습니다.
  * 밴드의 웹 FrontEnd를 Modern Framework 기반으로 전환 진행중
  * 신규 글로벌 커뮤니티 서비스를 React 기반으로 구성중

## 우리의 목표는 이렇습니다.

* 모두가 변화에 열려 있는 팀
* 최신 기술에 항상 준비되어 있는 팀 &#x20;
* 어떤 신규 서비스를 기획 하더라도 기민 하게 대응 할 수 있는 팀
* 개개인의 실험 정신을 맘껏 발휘 할 수 있는 팀
* 상대방의 의견에 항상 열려있고 긍정적으로 서로 소통하는 팀&#x20;
* 항상 성장하고 발전하는 팀
* 일하는 재미가 있는 팀&#x20;

## 아래와 같은 경험이 있는 분이면 더욱 좋습니다.

* React, Vue, Angular 등의 최신 Modern Framework 기반하에서 경험이 풍부하신분
  * 사용법만이 아닌 해당 framework의 원리와 의도를 충분히 이해하고 계신분
  * framework 사용 최적화에 관심이 많으신 분
* Webpack등 다양한 빌드 도구를 통해 시스템 구성 및 최적화에 경험이 많으신 분
* FrontEnd뿐만 아니라 BackEnd에 대한 기본 지식을 갖추신 분
* FrontEnd 개발 방법론에도 관심이 많아 항상 최적화에 대한 고민을 하시는 분
