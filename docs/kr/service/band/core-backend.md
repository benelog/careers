# Core Backend

### 입사하게 되면 담당할 업무입니다 

* 채팅 플랫폼
  * 대규모 인원이 참여 하는 라이브 채팅 뿐만 아니라,  소규모 인원의 그룹채팅, 1:1 채팅까지 채팅에 대한 모든 솔루션을 제공합니다. &#x20;
  * 사내 네이버 오픈 채팅, 사외의 위버스 라이브 채팅을 위한 플랫폼을 제공합니다.  BTS 라이브 처럼, 여러 국가에서 많은 인원이 동시접속하여 대규모의 트래픽이 발생하는 상황에서도 안정적으로 서비스 되는 채팅 플랫폼을 만듭니다. &#x20;
  * BAND, CAFE  서비스의 채팅 개발을 담당합니다.&#x20;
* 설문 플랫폼&#x20;
  * 사내외 여러 서비스에서 필요로 하는 설문 기능을 위한 플랫폼을 제공합니다.  쇼핑 '소비자조사솔루션'의 설문 조사 도구로써 플랫폼이 제공되고 있습니다.

### 업무를 수행하기 위해 필요한 자격 요건입니다 

* 웹 서비스를 위한 Backend API 를 개발한 경험이 있으신 분
* 동료들과 원활한 의사 소통이 가능하신 분

### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다 

* 담당할 업무에서 사용할 스킬
  * **Infra**: Linux, Docker
  * **Web Server**: Nginx, Apache, Tomcat
  * **Web Framework**: Spring Framework/Spring Boot, Actor
  * **DB**: MySQL
  * **Cache**: Arcus(Memcached), Redis
  * **Data Processing**: Kafka
* 담당할 업무와 관련된 경험
  * 커뮤니티 서비스 개발 경험
  * 대용량 트래픽 처리 시스템 개발 경험
  * 장애 예방/장애 전파 방지를 위한 시스템/모니터링 개발 경험
  * 채팅 시스템 개발 경험
  * 대규모 데이터 처리 경험
