# Android

## BAND, CAFE 등 커뮤니티 서비스 Android 앱개발

### 이런일을 맡고 있어요.

우리는 [밴드](https://band.us), [카페 ](https://cafe.naver.com)서비스등 커뮤니티 관련 서비스의 안드로이드앱 개발을 담당하고 있으며 관련 서비스의 신규 기능 개발, 기존 기능 개선, 원활한 운영, 최신 기술적용을 수행하여 서비스 성장에 기여하는 것을 목표로 합니다.

### 입사하게 되면 이런걸 경험할 수 있어요.

* 밴드, 카페등 커뮤니티 서비스 관련 다양한 기능 개발을 경험할수 있어요.
* 글을 쓰고, 보여주는 가장 기초적이지만 핵심적인 기능 개발을 경험할수 있어요.&#x20;
* 내가 만든 코드가 다양한 사용자 환경에서 어떤 문제를 일으킬수 있는지 경험할 수 있어요.
* 올해 구글 IO에서 나온 최신 트랜드를 바로 적용해보는 경험을 할 수 있어요.
* Compose 기반 디자인 시스템이 적용된 UI 개발을 경험할수 있어요.
* 레거시 코드를 최신 기술이 적용된 코드로 리팩토링하는 경험을 할 수 있어요.
* 애자일 방법론에 기반한 효율적인 개발 문화를 경험을 할 수 있어요.

### 업무를 수행하기 위해 필요한 자격 요건입니다.

* Kotlin,Java 프로그래밍에 대한 기본 지식을 가지신 분.
* 객체 지향 개발에 대한 기본 지식을 가지신 분.
* Android 주요 컴포넌트를 이해하고 사용한 경험이 있으신 분.

### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다.

* AAC, MVVM, Clean Architecture 기반으로 개발 경험이 있으신 분.
* Rx(or Flow), Coroutine 같은 비동기처리에 능숙하신 분.
* Dagger, Hilt와 같은 DI 사용 경험이 있으신 분.
* Compose 사용 경험이 있으신 분.
* 글 편집기, 피드등 복잡한 UI 개발 경험이 있으신 분.
* 무엇보다 중요한 건 성장하고 싶다는 열정이 있으신 분.^^
