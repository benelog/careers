# Global Community Dev BackEnd

### 입사하게 되면 담당할 업무입니다 

* 밴드
  * 한국에서 많은 사용자가 사용하는 서비스이고, 미국 / 일본의 사용자도 적지 않게 사용 중으로, 매년 해외 사용자가 잘 성장하고 있습니다.
  * 사용자가 많은 만큼, 대용량 트래픽 / 데이터 처리를 수행합니다.
  * 밴드만의 회원 , 인증 체계를 운영하고 있어 인증 설계, 운영을 경험해 볼 수 있습니다.
  * 새소식, 글 / 댓글 / 대댓글 등의 기본 커뮤니티 기능 이외에, 캘린더 전문 서비스에 준 하는 캘린더 기능과, 퀴즈 / 설문 등의 기능도 제공하고, 위치 공유, 밴드별 용량 쿼터 처리 등, 다양한 기능들을 제공하고 있어 커뮤티니 관련된 다양한 기능을 경험해 볼 수 있습니다.
  * 미국 사용자가 300만명이 넘는 서비스로, 해외 사용자를 위한 인프라 구성 및, 그에 따른 처리를 담당할 수 있습니다.
* 커뮤니티 플랫폼
  * 밴드에서 제공하는 많은 커뮤니티 기능들을 개발했던 경험을 살려, 다양한 곳에 쉽게 붙일 수 있는 커뮤니티 플랫폼을 신규로 구축하고 있습니다.
  * 국내 뿐 아니라, 글로벌 출시도 염두해 두고 있습니다.

### 업무를 수행하기 위해 필요한 자격 요건입니다 

* 웹 서비스를 위한 Backend API 를 개발한 경험이 있으신 분
* 동료들과 원활한 의사 소통이 가능하신 분
* 논리적인 사고

### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다 

* 담당할 업무에서 사용할 스킬
  * **Infra**: Linux, Docker
  * **Web Server**: Nginx, Apache, Tomcat
  * **Web Framework**: Spring Framework/Spring Boot
  * **DB**: MySQL, Vitess
  * **Cache**: Arcus(Memcached), Redis
* 담당할 업무와 관련된 경험
  * 커뮤니티 서비스 개발 경험
  * 대용량 트래픽 처리 시스템 개발 경험
  * 장애 예방/장애 전파 방지를 위한 시스템/모니터링 개발 경험
  * 대규모 데이터 처리 경험
