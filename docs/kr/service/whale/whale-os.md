# Whale OS

## Whale OS 개발

#### 이런 일을 맡고 있어요

오픈소스 크로미움 OS 기반으로 만들어진 WhaleOS는 단순히 오픈소스를 가져다 사용하는 것이 아닌 로그인, 브라우저, 앱스토어, 업데이트, 클로바 등 대부분의 주요 서비스들을 직접 개발하여 적용한 OS로, Whale OS 기반 웨일북 위에서 동작하는 OS를 개발하고 있습니다.

* 웨일 OS의 리눅스 커널부터 주요 웹 서비스들까지 다양한 기술 스을 경험 수 있습니다.
* 좋은 아이디어를 직접 제품에 적용할 기회가 있습니다.
* 업무 이외에도, 오픈소스 기여에 함께 참여할 기회가 있습니다.​

**입사하게 되면 담당할 업무입니다**

* Whale OS BSP\(Board Support Package\) 개발
* OS 레벨 API 개발 및 웹 앱 생태계 지원
* 로그인, 브라우저, 클로바 등 컴포넌트 개발
* 안드로이드, 리눅스 앱 지원을 위한 런타임 개발

**업무를 수행하기 위해 필요한 자격 요건입니다**​

* C++ 언어 기반의 개발 경험
* 웹 브라우저 동작에 대한 기본적인 이해
* 객체지향 개발, 리팩토링, 자동화 테스트에 대한 기본 지식
* 하드웨어 자원\(CPU/GPU/DSP/ASIC\) 및 컴퓨터 아키텍처에 대한 이해 및 배경지식

**추가로 보유하고 있다면 좋을 스킬 셋이나 경험입니다**​

* 웹 OS 등 자체 OS 개발 경력
* WebKit 또는 Chromium 등 오픈 소스 활동 경험
* 대규모 코드 베이스 개발 경험
* 적극적 태도로 다양한 방식을 통해 주어진 문제를 해결하는 데 능숙한 사람 
* 새로운 기술을 배우는 것에 대해 거리낌이 없고 공유하는 것에 즐거움을 느끼는 사람
