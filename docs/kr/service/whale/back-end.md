# Back-end

## Whale 브라우저 백엔드 서비스 개발

#### 이런 일을 맡고 있어요

Whale Server 팀은 웨일브라우저에 필요한 백엔드를 개발합니다.

* 새로운 웹 기술과 브라우저, OS 백엔드 개발
* 웨일 엔터프라이즈 서비스 개발
* 동기화, 보안, 확장앱, 원격회의 등 다양한 웹 서비스 개발

#### 입사하게 되면 담당할 업무입니다

* 엔터프라이즈 웨일브라우저 API 및 웹서비스 개발
* Whale OS용 API 및 웹서비스 개발
* 웹기반 화상회의 서버 개발
* 앱 스토어 개발
* 브라우저 및 서비스 데이터 분석

#### 업무를 수행하기 위해 필요한 자격 요건입니다

* 프로그램 설계 디버깅, 자료구조, 알고리즘에 대한 넓은 이해가 가능하신 분
* Java / node.js / go / Python 중 1개 이상의 개발에 익숙하신 분
* 자기주도적인 업무태도와 동료와의 협업에 적극적이신 분
* 단순한 기술활용보다는 깊이있는 이해를 추구 하시는 분

#### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다

* Spring Framework을 이용한 경험이 풍부하신 분
* 대용량 트래픽/데이터 개발에 대한 경험과 노하우가 풍부하신 분
* 대용량 Back-end 서비스 시스템 개발 경험
