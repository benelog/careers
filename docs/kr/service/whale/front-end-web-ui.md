# Front-end

## 네이버 웨일 WebUI 개발

#### 이런 일을 맡고 있어요

자유로운 웹 환경을 위한 새로운 브라우저를 만드는 조직으로 다음과 같은 기회가 있습니다.

* 최신 웹 엔진 기반 환경에서 다양한 애플리케이션과 서비스를 개발할 수 있습니다.
* 새로운 웹 기술과 브라우저, OS 개발을 경험할 수 있습니다.
* 표준 웹 스펙 개발에 참여할 수 있습니다.

#### 입사하게 되면 담당할 업무입니다

* 웨일 브라우저 / 웨일 OS WebUI 개발
* 웨일 웹 서비스 개발 \(스토어, 스크랩북, 마이 액티비티, 웨일온, 웨일 스페이스 등\)
* 웨일 브라우저 확장 앱, 웨일 OS 앱 개발

#### 업무를 수행하기 위해 필요한 자격 요건입니다

* Vue.js, Polymer, React, Lit-Element 등 모던 프레임워크 활용 FE 개발 경험
* 모바일을 포함한 다양한 환경에서의 FE 개발 경험
* 브라우저 동작에 대한 이해

#### 추가로 보유하고 있다면 좋을 스킬 셋이나 경험입니다

* 새로운 업무나 기술을 배우는데 거리낌 없으신 분
* 동료와 협업하면서 업무를 함께 진행할 수 있으신 분
* 아이디어가 풍부하고 새로운 것을 만드는 것을 좋아하시는 분
* 자신의 기술과 경험을 잘 나눌 수 있으신 분
