# Web Engine

## 웨일 브라우저 웹 컴포넌트와 웹 엔진 그리고 XWhale 개발

#### 이런 일을 맡고 있어요

웨일 브라우저 5개 플랫폼 \(Windows, macOS, Linux, Android, iOS\)의 브라우저 컴포넌트 \(로그인, 북마크, 싱크 등\) 및 웹 엔진 개발 그리고 네이버 앱등에서 사용하고 있는 XWhale framework를 개발하고 있습니다.

* 데스크탑 / 모바일을 통합하는 큰 기술 플랫폼으로 자리 잡고 나아가 새로운 비즈니스 플랫폼을 만들고 있습니다.
* 팀 내 오픈소스 활동 중이신 분들이 있어, 오픈소스에 활동을 시작하는 데 도움을 얻을 수 있습니다.
* 주로 C++로 작성된 크로미움 프로젝트를 다루게 되고, iOS, Android, Frontend 코드를 직간접적으로 경험할 수 있습니다.
* 팀 내의 웹 엔진 전문가들로부터 많은 기술적인 도움을 받을 수 있습니다.

**입사하게 되면 담당할 업무입니다**

* 웨일 브라우저에서 사용하고 있는 주요 컴포넌트\(로그인, 싱크 등\) 개발
* 웹 엔진 관련 기능, 보안, 성등 최적화 등 웹 엔진 기반 주요 기술 개발
* 3000만 명 이상 사용하는 세계 유일무이한 XWhale framework를 개발
* 업무 이외에도, 오픈소스 기여에 함께 참여할 기회

**업무를 수행하기 위해 필요한 자격 요건입니다**​

* C++ 언어 기반의 개발 경험
* 웹 브라우저 동작에 대한 기본적인 이해
* 객체지향 개발, 리팩토링, 자동화 테스트에 대한 기본 지식

**추가로 보유하고 있다면 좋을 스킬 셋이나 경험입니다**​

* WebKit 또는 Chromium 등 오픈 소스 활동 경험
* 대규모 코드 베이스 개발 경
* 적극적 태도로 다양한 방식을 통해 주어진 문제를 해결하는 데 능숙한 사람 
* 새로운 기술을 배우는 것에 대해 거리낌이 없고 공유하는 것에 즐거움을 느끼는 사람
