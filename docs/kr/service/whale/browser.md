# Browser

## 웨일 브라우저 UI 개발

#### 이런 일을 맡고 있어요

더 나은 사용자 경험을 전달하기 위해, 유려한 디자인을 실체화하고 사용자 친화적인 기능을 만들고 있습니다. ​

* 브라우저 UI 기능을 개발하면서, 브라우저 전반에 대한 기술을 배울 수 있습니다.
* 다양한 플랫폼에서의 개발 경험을 얻을 수 있습니다.
* 좋은 아이디어를 직접 제품에 적용할 기회가 있습니다.
* 업무 이외에도, 오픈소스 기여에 함께 참여할 기회가 있습니다.​

**입사하게 되면 담당할 업무입니다**

* 브라우저 UI 기능 개발 및 디자인 적용 \(C++\)
* 브라우저 기반의 주요 제품군 UI 기능 개발 \(웨일온, WhaleOS, Whale Enterprise\)
* 플랫폼별 호환성 이슈 대응 \(Windows, Mac, Linux, WhaleOS\)
* 코드 리베이스

**업무를 수행하기 위해 필요한 자격 요건입니다**​

* C++ 언어 기반의 개발 경험
* 웹 브라우저 동작에 대한 기본적인 이해
* 기본적인 소프트웨어 개발 기술 \(객체지향 개발, 리팩토링\)

**추가로 보유하고 있다면 좋을 스킬 셋이나 경험입니다**​

* UI 개발 경험
* 오픈소스 기여 경험
* 실험적/도전적인 개발자
