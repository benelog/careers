# Data

### 이런일을 맡고 있어요

네이버 글로벌 광고 플랫폼을 담당하고 있는 조직으로 다음과 같은 기회를 가질 수 있습니다.

* 글로벌 광고 플랫폼 개발 경험을 통해 향후 애드테크 도메인의 전문가로 성장
* 네이버, 라인 등에서 발생하는 막대한 트래픽을 처리하는 대용량 시스템 운영 경험

### 입사하게 되면 담당할 업무입니다

* 광고 지표 수집 및 통계, 어뷰징, BA, 리포트 API 등 데이터 처리에 관한 모든 것을 담당합니다.

### 업무를 수행하기 위해 필요한 자격 요건입니다

* 대용량 데이터 처리에 대한 기본 지식과 파이프라이닝 설계 경험이 있으신 분 
* Hadoop Eco System 중 2개 이상 능숙하게 다루는 제품이 있는 분 
* 데이터에 기반한 문제 해결에 익숙한 분

### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다

* 대용량 트래픽/데이터 개발에 대한 경험과 노하우가 풍부하신 분 
* 머신러닝, 데이터 분석, 통계 지식을 보유하신 분
