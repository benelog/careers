# Back-end

### 이런일을 맡고 있어요

* 광고 데이터 처리를 위한 플랫폼 운영 및 파이프라인 개발 
* 광고 시스템의 데이터 분석을 통한 abnomaly detection 
* 국내 최대의 검색/디스플레이 광고 플랫폼 개발 
* 글로벌 광고 플랫폼 연계 및 ad-network 개발 
* 네이버 쇼핑 도메인의 광고 서비스 개발

### 입사하게 되면 담당할 업무입니다

* 광고 데이터 파이프라인 개발 
* 시계열 Abnormaly Detection 연구 
* AI/ML 모델 서빙 시스템 개발 
* 키워드 관리, 매칭, 랭킹, 과금 등 전송 로직 개발 
* 광고 컨텐츠 노출/탐색 서비스 개발 
* 광고 개인화 추천 시스템 개발\(DPA, DCO\) 
* 매체향 광고 서버 및 매체 수익화 시스템 개발

### 업무를 수행하기 위해 필요한 자격 요건입니다

* 광고 비지니스로직에 대한 이해가 높거나 관심이 많은 분 
* Hadoop, Spark, kafka 등을 활용한 대용량 데이터 처리 경험 
* Java / Kotlin / Go 중 1개 이상의 개발에 익숙하신 분 
* 랭킹/추천 관련 논문 습득, 이해 및 빠른 프로토타이핑에 능숙한 분 
* 최근 2년 이상 Spring Framework 개발 경험이 있으신 분 
* RDBMS, NOSQL 에 대한 이해와 개발 경험이 있으신 분 
* 비즈니스 또는 데이터에 기반한 문제 해결에 익숙하신 분

### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다

* 대용량 real-time stream processing \(kafka & spark stream, etc\) 경험 
* 시계열 Abnormaly Detection에 대한 경험 
* 코드리뷰, TDD, CI/CD 강화 등의 개발/운영 가이드 확립 경험 
* ES 커스터마이징 경험과 높은 이해도 · 오픈소스 활용에 대한 경험 
* Docker & Kubernetes에 대한 지식 및 운영 경험 
* Ad Tech 관련 개발 경험
