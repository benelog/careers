# Front-end

### 이런일을 맡고 있어요

네이버와 라인의 글로벌 광고 플랫폼과 쇼핑광고, 여행서비스, Flyer 서비스등 버티컬 서비스의 개발을 담당하고 있습니다. 글로벌한 개발 환경에서 트랜디하고 다양한 FE 개발을 깊이있게 경험할 수 있는 기회가 있습니다. 특히나 네이버와 라인에서 발생하는 막대한 트래픽에 대응할 수 있는 시스템을 개발하는 도전적인 경험도 할수 있습니다.

### 입사하게 되면 담당할 업무입니다

* 글로벌 광고 플랫폼의 FE/BFF 개발 
* 네이버 쇼핑 버티컬 광고 플랫폼의 FE/BFF 개발
* 네이버 여행 서비스의 FE/BFF 개발 
* 라인 Flyer 서비스의 FE/BFF 개발

### 업무를 수행하기 위해 필요한 자격 요건입니다

* Javascript 개발 경력 2년 이상 
* HTML, CSS 등 W3C 웹 표준 규격에 대한 이해 
* React, Vue.js 등 최신 Framework 을 이용한 개발 경험

### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다

* NodeJS, Typescript, GraphQL 등의 실무 적용 경험 
* SPA\(Single Page Application\), SSR\(Server Side Rendering\) 개발경험 
* RDBMS, NOSQL 를 이용한 개발 경험 
* 지식을 적극 공유하며 함께 성장하려는 협업 마인드
