# PWE앱

PWE앱은 캘린더앱, 메일앱, 메모앱을 기준으로 개인화 기반 서비스앱을 만들어 나가고 있습니다.

사용자 일상의 개인 데이터를 관리하고, 저장하고, 확인하는 사용자 밀착형 앱을 만들어나가는 특성상 새로운 사용자 경험 개선과 최적화된 사용성 제공에 중점을 두고 있습니다.

또한 동적 운영(LiveOps), 품질(Vitals), CI/CD 등 DevOps 관점에서의 기술적 체계를 정립해 나가고 있고, Declarative UI, Cross Platform등 다양한 기술적 시도를 통하여 플랫폼 트렌드에 맞춰 서비스앱을 발전시켜 나가는 것을 목표를 하고 있습니다.
