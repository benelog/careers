# iOS



네이버앱 iOS 개발 조직은 iOS 플랫폼 기반의 네이티브/웹 기반의 서비스 개발과 개발 환경 개선을 위한 업무에 참여하게 됩니다.&#x20;

#### # 이런 일을 하게 됩니다.  

* 네이버앱 개발
  * 네이버 서비스 기능 개발&#x20;
  * 인앱 브라우저 기능 고도화&#x20;
  * Watch 서비스 개발&#x20;
* 운영 개발
  * CI/CD 개발&#x20;
  * 동적 운영(LiveOps) 전략과 기능 고도화&#x20;
  * DevOps&#x20;

#### # 일을 하기 위해 다음과 같은 경험과 이해가 필요합니다. 

* Swift 개발 경험
* iOS Framework 및 View 구조에 대한 이해&#x20;
* 다양한 UI 및 인터랙션 개발 경험&#x20;
* 새로운 기술에 대한 관심&#x20;

#### # 이런 경험과 관심이 있으면 더 좋아요. 

* 아키택처 패턴에 대한 이해&#x20;
* 앱 성능 Profiling에 대한 이해
* WebView에 대한 이해&#x20;
* Objective-C 개발 경험
* 앱 모듈화 경험&#x20;
* CI/CD 개선에 대한 관심&#x20;
