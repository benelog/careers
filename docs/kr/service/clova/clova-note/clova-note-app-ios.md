# CLOVA Note App (iOS)

r![](/assets/kr/image%20(55).png)

## 소개

CLOVA Note는 AI(음성인식) 기술을 이용해 음성을 텍스트로 변환하여 음성 기록을 편리하게 사용, 관리할 수 있는 서비스입니다.

CLOVA Note iOS 팀에서는 Apple Platform의 Native App을 개발하여 사용자들이 좋은 경험을 할 수 있도록 노력하고 있습니다.

## 개발 업무

* CLOVA Note App (iOS) 개발

## 기술

* Swift
* SwiftUI
* Combine

## 우대 사항

* 다양한 Platform Application 개발 경험 (iOS, iPadOS, macOS, watchOS)
* UI / UX, 신규 기술에 대한 관심
* Audio 도메인 개발 경험
