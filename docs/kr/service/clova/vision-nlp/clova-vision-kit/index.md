# CLOVA Vision Kit

## 모집 개괄

세계 수준의 Hyper AI를 만들어나가는 NAVER Clova에서 Computer Vision의 세계적인 경쟁력 확보를 위한 대규모 인재 채용을 진행합니다.

연구자들과 개발자들이 매년 다양한 연구 뿐만아니라 실제 유저를 만나는 Clova Lamp, Face Sign, Payment eKYC, LINE Avatar, Vlive Autocam, Sport Video Classification 등의 프로덕트에 다양한 기여를 하고있습니다.

대량의 데이터와 인프라에서 현실 세계의 문제를 AI기반으로 해결하는 것을 넘어서, 실제 프로덕트로 만들고, 이를 Global 서비스에 적용해나가는 전반의 과정을 함께 해나갈 인재를 찾습니다.

* Computer Vision Researcher: 15명 \(세부 모집 부분인 하단을 참고\)
* Edge ML Research: 6명
* Android, iOS SDK 개발: 6명
* Data Engineer \(Media Data 구축 및 가공\): 3명
