# Data Engineer

#### 이런 일을 하고 있어요.

Vision AI개발을 위해 구축한 데이터를 보관하고 분석하고 관리하는 역할을 하는 조직입니다.

Data Driven AI 시대에 개발과 분석을 넘어서 vision product를 오너십있게 같이 만들어갈 인재를 찾습니다.

최근 AI의 트렌드는 "대량의 데이터"입니다. Image나 Video 데이터의 특성상 큰 용량과 다양한 정보를 담고있고, 이를 필요한 모델에 맞추어 찾아내고 가공해서 전달하는 역할을 하고있습니다.

* Image, Video를 효율적으로 관리하고 가공하는 업무

* data visualization 또는 insight extraction

데이터를 수집하고 가공하여 모델 학습에 제공하는 업무를 주로 하게됩니다.

그리고 모델을 통해 인입된 데이터를 저장하고, 낮은 정확도의 데이터를 선별하고, 어노테이션해서 모델을 개선하고 서비스를 운영하는 과정을 진행하는 역할이 됩니다.

꾸준하게 데이터를 보고 책임감있게 관리하고 더 효율적으로 제공하기위해 끊임없이 함께 고민할 인재를 찾습니다.

* 데이터 처리 시스템 설계, 구축, 운영 경험이 있으신분
* 통계적으로 메타 정보를 분석해본 경험
* AI 모델 품질 평가 경험
* 이미지, 비디오 데이터의 crawling, 구매 등의 다양한 방법을 통한 수집, 관리 경험이 있으신분
