# Avatar AI

#### 조직소개

네이버 CLOVA의 Avatar 조직은 Virtual Avatar를 생성하는데 필요한 요소 기술을 연구 개발합니다.

사람의 자세 및 손 동작을 예측하기도 하고 얼굴\(표정 및 입\) 및 모션을 생성하며, NAVER와 LINE의 다양한 서비스에 제공하고있습니다.

#### 주요 역할 <a id="ClovaVisionKitJD&#xC815;&#xB9AC;-&#xC8FC;&#xC694;&#xC5ED;&#xD560;.2"></a>

* RGB 이미지 기반 2D/3D Human Pose\(Body, Hand\) Estimation 모델 개발
* RGB 이미지 기반 Hand Gesture Recognition 모델 개발

#### 자격요건 <a id="ClovaVisionKitJD&#xC815;&#xB9AC;-&#xC790;&#xACA9;&#xC694;&#xAC74;.2"></a>

* Pose 혹은 Gesture Recognition 관련 개발 경험을 보유하고 계신 분
* 주요 머신러닝 프레임워크에 대한 이해가 깊으신 분 \(PyTorch, TensorFlow, Keras 등\)

#### 우대 조건 <a id="ClovaVisionKitJD&#xC815;&#xB9AC;-&#xC6B0;&#xB300;&#xC870;&#xAC74;.1"></a>

* Computer Vision 및 Graphics top-tier 학회에 논문 게재 경험을 보유하고 있는 분
* 경량화 모델 개발 경험을 보유하고 있는 분
* 머신러닝을 활용한 서비스 개발 경험을 보유하고 있는 분
* C/C++, Python 등 숙련된 개발 능력이 있는 분
