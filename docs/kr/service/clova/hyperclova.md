# HyperCLOVA Engineering

### 소개

하이퍼스케일 AI 를 활용하여 쉽게 서비스를 제작할 수 있는 환경을 만들고 있습니다.&#x20;

* HyperCLOVA 를 사용해 보고, 갖고 있는 데이터로 튜닝하여 실 서비스 연동까지 지원해 주는 클로바 스튜디오    &#x20;
* 대화 시나리오를 작성하고 빌드하여 라인, 톡톡, 전화 등 다양한 경로로 서비스할 수 있는 클로바 챗봇

한국/일본/대만 글로벌 서비스 경험 뿐만 아니라, 하이퍼스케일 AI 를 활용한 새로운 서비스 경험도 쌓으실수 있습니다.

### 담당 업무  &#x20;

* 누구나 쉽게 언어 모델을 활용한 서비스를 만들 수 있는 플랫폼 개발
  * HyperCLOVA 를 사용하여  모델을 만들고, 튜닝하며, 서비스 할 수 있는 플랫폼 제공
  * 다양한 목적의 대화 모델을 구축할 수 있는 챗봇 빌더 개발
  * 여러 모델, API, 코드를 엮어 쉽게  서비스를 만들 수 있는 환경 제공
* 효율적인 개발과 신뢰할 수 있는 운영을  위한 프로세스 및 도구  개발
  * Micro Service Architecture 에서 개발, 배포, 운영을 돕기 위한 프로세스 수립 및 도구 개발         &#x20;
* 점진적 개선을 위한 데이터  파이프라인 구축 및 통계 수집

### **자격 요건**

* Kotlin/Scala/Java 를 사용한 서버 개발이 가능 하신 분
* RDBMS, NoSQL, Message Queue 를 활용하여 안정적이고 효율적으로 데이터를 관리하실 수 있는 분
* 인터넷 기반 프로토콜에 대한 이해를 바탕으로 구현 및 문제 해결이 가능하신 분
* Linux / Kubernetes 기반으로 자동화된 build 및 deploy 환경을 구축하실 수 있는 분

### **우대 사항**

* Spring 혹은 Akka 에 익숙하신 분  &#x20;
* webflux / coroutine 등을 활용하여 Non-Blocking 기반 서버 개발 경험이 있으신 분
* 고가용성의 확장 가능한 시스템을 설계하고 운영해 보신 분
* Unit Test / Integration Test 등을 활용하여 안정적으로 서비스를 개선하시는 분
* Script 실행을 위한 Sandboxing 환경을 구축해 보신 분   &#x20;

### **담당 업무** &#x20;

* 서비스 제작을 쉽게 도와주는 웹 기반의 도구 개발
  * HyperCLOVA 모델 사용을 쉽게 테스트 해 볼 수 있는 editor 기반 툴 개발
  * HyperCLOVA 튜닝을 위한 파일 업로드 컴포넌트 및 관련 기능 개발
  * 다양한 목적의 대화를 설계하고, 외부 시스템과 연동을 할 수 있는 챗봇 빌더 개발
  * 대화 flow 를 그릴 수 있는 대화 빌더 개발
* 대화 사용을 쉽게 만들어 주는 채팅 웹 개발
  * 쉽게 웹페이지에 채팅 창을 붙일 수 있는 웹챗 개발

### **자격 요건**

* React 와 같은 SPA 프레임워크를 사용하여 웹페이지 개발이 가능하신 분
* HTML, CSS, 최신 ECMAScript 에 대한 이해가 깊으신 분
* Component 주도 개발을 통해 프로젝트 효율을 높일 수 있는 분
* 인터넷 기반 프로토콜/기술에 대한 이해가 있으신 분 (OAuth / HTTP / TCP-IP 등)

### **우대 사항**

* TypeScript 를 활용한 개발 경험이 있으신 분
* Redux 와 같은 State Container 에 대한 이해가 있으신 분
* Node.js 를 통해 간단한 WAS 서버 작성이 가능하신 분
* Kubernetes 를 활용한 배포 자동화 경험이 있으신 분
