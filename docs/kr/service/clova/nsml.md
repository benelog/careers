# ML 연구 플랫폼 NSML 개발

### 소개

NSML팀은 CLOVA의 ML 연구 플랫폼을 만들고 제공합니다.&#x20;

간단한 실험부터 실제 문제를 푸는 대규모 모델 학습까지, ML 연구원의 생산성과 잠재력을 극대화하는 것이 우리의 미션입니다. Kubernetes와 gRPC 등의 백엔드 기술을 이용하여 신뢰성과 가용성을 모두 갖춘 ML 연구 플랫폼에 대한 풍부한 경험을 제공하고 있습니다.

### **역할**

* NSML 서비스 BE 개발
* NSML 서비스 FE 개발
* NSML 서비스 SRE
* GPU 클러스터 운영

### **자격요건**

* 프로그램 설계 디버깅, 자료구조, 알고리즘에 대한 넓은 이해가 있는 분
* Go / Python / JavaScript / TypeScript 중 1개 이상 언어에 익숙한 분
* BE 혹은 FE 개발에 전문성을 갖춘 분
* MLOps에 관심있는 분

### **우대사항**

* 시스템 설계 전문성
* Kubernetes 전문성
* 데이터 시각화 전문성
* React 개발 경험
* 복잡한 대규모 서비스나 클라우드 서비스 개발 및 운영 경험
