# Document Intelligence

### **Document Intelligence**

*   **팀 소개**

    문서를 가장 많이 생산하고 소비하는 곳은 업무 공간, 우리가 근무하는 기업 환경입니다. 기업 환경에서 생산되는 대량의 비정형의 문서에 포함된 정보를 작업자들이 이해하기 쉬운 형태로 분석하고 가공해 주는 AI 제품을 초대규모 언어 모델인 HyperCLOVA를 기반으로 연구/개발하는 팀입니다.
