# DATA AI

### 조직소개&#x20;

Data AI 팀에서는 머신러닝 / 딥러닝 기술 연구에 필요한 학습 및 평가 데이터셋 구축을 위해 데이터 어노테이션 플랫폼을 운영하고 있습니다. 저희와 함께 하시면 데이터 관리 시스템 운영 및 어노테이션 플랫폼 개발 등 다양한 영역에서 커리어를 쌓을 수 있습니다.

### 1. 학습 데이터 생성 플랫폼 개발 (프론트)

#### 역할 및 필요역량

* 학습 데이터 생성을 위한 플랫폼 개발 (프론트)
* 자기주도적 문제 해결 능력 및 협업에 필요한 커뮤니케이션 능력

#### 필수조건     &#x20;

* 중급 이상의 자바스크립트 활용 능력
* 프론트 엔드 개발 경험
* 졸업예정자(학기가 남아있어도 근무 가능하면 문제 없음)

#### 우대요건

* 리눅스 환경에서의 개발 경험
* 프로젝트(개인/인턴 등) 완성 경험
* Python 활용 능력
* 백엔드 개발 경험
