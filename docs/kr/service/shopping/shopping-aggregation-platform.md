# 쇼핑검색서비스

### 이런 일을 하고 있어요.

스마트스토어 뿐만 아니라 외부 제휴몰로부터 대량의 상품 정보과 상품 메타 정보들을 수집하고 정제하는 파이프라인을 통 DB구축을 진행하고 있습니다.&#x20;

이렇게 구축된 데이터는 최신 AI/ML기술을 통해 동일한 상품을 묶어주는 카탈로그를 생성하는 기반 데이터로 이용되고 있으며,  데이터 분석을 통해 사용자들의 검색 시나리오를 풍부하게 만들어줄 추천 데이터들을 생성하고 있습니다.

이런 모든 과정을 통해 구축된 데이터는 사용자에게 최고의 검색 경험을 제공해 줄수 있는 UI로 구성된 웹서비스를 통해 제공함으로써 국내 최고의 쇼핑 검색 서비스를 만들어가고 있습니다. 이렇게 국내에서 축적된 기술과 노하우를 바탕으로 글로벌 서비스로의 확장을 준비하고 있습니다.

## 입사하게 되면 담당할 업무입니다.

한국/글로벌을 대상으로 쇼핑검색 플랫폼을 개발합니다.

* 상품 및 정보성 메타 데이터 수집 파이프라인 개발
* 카탈로그 개발
* 데이터 정제 파이프라인 개발
* 검색/가격비교 웹 서비스 개발
* 실시간 대용량 트래픽 처리 시스템 개발
* 분산 RDBS 기반의 어플리케이션 개발
* 빅데이터 플랫폼 운영
* 빅데이터 수집 및 분석 시스템 개발
* MLops 구축
* AI/ML 모델링 서비스 적용

### 업무를 수행하기 위한 요건입니다.

### Front-End

#### **기본 요건**

* Typescript + React.js 기반 개발 경험
* React lifecycle을 이해하고 hook을 사용해본 경험
* 웹 / 모바일 서비스의 일반적인 아키텍쳐 이해 및 RESTful API 설계

#### **우대 요건**

* Next.js 기반 SSR 개발 경험
* Node.js 기반 API 개발 경험
* k8s 기반 서비스 배포/운영 경

### Back-End

#### 기본 요건

* 백앤드 서버 및 배치 프로세스 개발 업무 경험
* Spring framework 기반의 어플리케이션 개발 경험
* RDBMS, NoSQL를 사용한 서비스 개발 경험
* 스트림 기반 데이터 처리 및 컨테이너 운용 경험
* 신기술 센싱과 분석을 즐기시는 분

#### 우대 요건

* k8s 등 컨테이너 기반 서비스 운영 경험
* kafka를 활용한 대용량 메세지 스트림 처리 경험
* ElasticSearch 클러스터 운용 및 서비스 적용 경험
* 분산 RDBMS를 서비스에 적용 경험

### Data Engineering

#### **기본 여건**

* Hadoop 및 Hadoop eco system 이해
* Spark / Spark streaming 개발 역량 보유
* Kafka 클러스터에 대한 이해, 활용 개발 역량 보유
* NoSQL 이해 및 서비스 적용 경험

#### **우대 요건**

* Hadoop 클러스터 운영 경험
* 대용량 데이터 파이프라인 구축 경험
* 빅데이터 분석 및 데이터 visualization 경험
* Kudu / Impala 이해 및 서비스 적용 경험

### AI / ML

#### 기본 여건

* AI/ML에 대한 배경 지식과 경험 보유
* 최신 논문을 수시로 이해하고 공유하며 토의할 수 있는 분

#### 우대 요건

* 다양한 데이터, 모델들을 앙상블/멀티모달하는 복합 과제에 경험 보유
* AI 기술과 더블어 E2E 엔지니어링을 위한 기술 보유
* kubeflow 구축 및 운영 또는 MLops 관련 기술 경험 보유
* ML 관점에서 CI/CD/CT 환경을 구성하고 운영 경험 보유
