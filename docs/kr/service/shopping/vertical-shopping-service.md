# 버티컬쇼핑서비스

### 이런 일을 맡고 있어요.

네이버 버티컬 쇼핑 서비스는 백화점/아울렛/장보기 서비스와 같은 O2O 쇼핑서비스 부터 디자이너, 키즈, 펫, 아트까지 다양한 버티컬 분야의 특성을 쇼핑서비스와 접목하여 **"둘러보는 재미"**&#xB97C; 줄수 있는 서비스를 만드는 것이 조직의  목표입니다.&#x20;

쇼핑 업계의 트렌드를 기민하게 파악하고 네이버 관점으로 해석하여 새로운 서비스를 개발하는 것이 주역할이기 때문에, 네이버에서 시작하는 새로운 쇼핑서비스를 만들어내는 **신규 쇼핑 서비스의 산실** 이라고 할 수 있습니다.\
이렇게 시작한 서비스로 네이버 쇼핑의 O2O 시작을 열었던 **백화점/스타일 윈도,** 제철 농수산물을 산지에서 바로 구매하여 싱싱하고 빠르게 배달받을 수 있는 **산지직송** 서비스, 홈플러스/GS프레시몰/농협하나로마트 등 대형 마트에서 장을 볼수 있을 뿐만 아니라, 집근처 백화점 식품관/동네시장에서 편리하게 장을 볼 수 있도록 서비스를 제공하는 **장보기** 서비스, 그리고 현재 라이브 쇼핑 업계의 절대 강자인 **네이버쇼핑 라이브**까지, 정말 다양하고 새로운 형태의 쇼핑 서비스가 버티컬 쇼핑 서비스에서 시작되었습니다.

특색있는 버티컬 쇼핑 서비스를 구현하기 위해 기술적으로는 네이버 쇼핑의 CMS(Content Management System) 역할을 담당하는 스마트스토어에서 생산한 다양한 판매자의 컨텐츠를 near-realtime 으로 연동하여 각 버티컬 특성에 맞게 데이터를 가공하고, 각 버티컬마다 가지는 고유한 속성들과 결합하여 잘 둘러볼 수 있는 최적의 데이터로 가공합니다. \
이렇게 정제된 데이터는 사용자의 피드백 데이터(방문/클릭/구매 등)와 결합하여 자동 또는 수동으로 구매자에게 제공되는 화면으로 구성되어 서비스되고, 여기서 발생하는 여러 사용자의 피드백 데이터는 다시 서비스의 품질을 개선하는데 사용하여 선순환을 만들어내고 있습니다.

## 입사하게 되면 담당할 업무입니다.

* 백화점/아울렛/펫/디자이너/푸드 등 O2O 기반의 다양한 쇼핑윈도 서비스 개발
* 장보기, MR., 럭셔리, 핫딜, 해외직구 등 다양한 버티컬 쇼핑 서비스 개발
* 새로운 유형의 쇼핑 서비스 발굴 및 개발
* 대용량 데이터 파이프라인 구축 및 데이터 처리 개발
* 콘텐츠 편집 도구 개발
* 서비스 지표 수집/분석 시스템 개발

## 업무를 수행하기 위한 요건입니다

### Back-End

#### 기본 요건

* Spring Framework의 구조와 매커니즘의 이해 또는 NodeJS의 구조와 매커니즘의 이해
* Java 또는 typescript 의 이해 및 개발 역량 보유
* 웹 / 모바일 서비스의 일반적인 아키텍쳐 이해 및 RESTful API 설계
* Test Framework를 사용한 Test 자동화 개발 이해
* RDBMS, NoSQL 를 사용한 서비스 개발
* Linux Server 환경에 대한 이해

#### 우대 요건

* Spring Security / Cloud / Boot / Batch 등 여러 Spring 환경 개발 경험
* Kafka, NiFI, StreamSets 등 메시징 미들웨어를 사용한 개발 경험
* Docker / Kubernetes 를 사용한 서비스 개발 및 운영 경험
* 대용량 트래픽 Backend 서비스 개발 경험

### Front-End&#x20;

#### 기본 요건

* 브라우저 구조 및 동작 원리 이해
* NodeJS의 구조와 매커니즘의 이해
* ES6, TypeScript의 이해 및 개발 역량 보유
* 웹 / 모바일 서비스의 일반적인 아키텍쳐 이해 및 RESTful API 이해
* Test Framework를 사용한 Test 자동화 개발 이해

#### 우대 요건

* react.js, vue.js, angular, next.js, webpack 등을 활용한 서비스 개발 경험
* graphql를 활용한 서비스 개발 경험
* 대용량 트래픽을 수용하는 Front-End 서비스 개발 경험

### Data Engineering&#x20;

#### 기본 요건

* Hadoop eco system에 대한 이해
* Spark, Spark stream 활용한 개발 역량 보유
* Kafka cluster에 대한 이해, 활용한 개발 역량 보유
* RDBMS, NoSQL 에 대한 이해

#### 우대 요건

* Secure hadoop에 대한 이해 및 활용한 서비스 개발 경험
* 대용량 Datapipeline 구축 및 운영 경험
* 대용량 데이터의 입수, 정제, 분석 개발 경험
