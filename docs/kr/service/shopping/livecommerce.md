# 라이브커머스

### 이런 일을 맡고 있어요.

**네이버 쇼핑라이브**는 휴대폰 하나로 간편하게 사용자와 판매자를 연결하는 국내 no.1 라이브커머스 플랫폼입니다.

네이버의 다양한 기술 플랫폼 (동영상 기술, 쇼핑, AI 등) 들의 기술이 집약된 서비스로 판매자분들께 쉽고 편리한 방송 송출, 관리 도구를 제공하고 시청자분들께는 더 좋은 소비 경험을 제공하기 위해 함께 고민하고 도전하실 개발자 분들을 찾고 있습니다.

최근에는 단순 상품 카테고리를 넘어 문화, 예술 콘텐츠까지 그 범위가 확대되고 있으며, 수십\~수백만명 이상의 동시 시청자들에게 뛰어난 품질과 안정적 서비스를 제공할 수 있는 인프라와 기술력과 경쟁력을 갖춰 나가고 있습니다.

라이브커머스 방송 송출 도구인 **쇼핑라이브 스튜디오 App**은 판매자가 쉽게 상품을 소개하고 고객들과 소통하기 위한 다양한 기능들 제공하여 언택트 시대에 어려움을 겪고 있는 많은 소상공인(SME)들의 판매 활동을 지원하고, 그뿐만 아니라 방송을 쉽게 관리하고 운영할 수 있도록 **쇼핑라이브 Web 관리툴**도 제공하고 있습니다.

앞으로 판매자와 시청자 모두가 만족할 수 있는 라이브커머스 플랫폼으로 한단계 도약을 준비하고 있습니다.

## 입사하게 되면 담당할 업무입니다.

라이브커머스 플랫폼을 개발합니다.

* 쇼핑라이브 웹 서비스 개발
* 방송 운영 관리도구 개발&#x20;
* 쇼핑라이브 App 개발 (iOS/Android)
* 쇼핑라이브 스튜디오 개발(iOS/Android)
* 라이브커머스 뷰어 SDK 개발 (iOS/Android/FE)
* 라이브커머스 솔루션 개발
* 라이브커머스 콘텐츠/혜택/이벤트 시스템 개발&#x20;
* MSA 구조 개발
* SRE 시스템 개발
* 통계 시스템 및 데이터 수집/분석 파이프라인 개발
* 실시간 트래픽 처리 시스템 개발
* 대규모 트래픽을 효율적으로 처리하고 Failover 가능한 아키텍쳐 설계 및 개발

### Back-End

#### &#x20;기본 요건

* Java / Spring Framework 기반의 어플리케이션 개발 경험
* 웹 / 모바일 서비스의 일반적인 아키텍쳐 이해 및 RESTful API 설계 경험
* 배치 프로세스 개발 경험
* 소규모 이상의 프로젝트 설계 경험
