# iOS

GLACE = Global Place

## 이런 일을 맡고 있어요.&#x20;

### LINE PLACE iOS

<figure><img src="../../.gitbook/assets/LINE_PLACE.png" alt=""><figcaption></figcaption></figure>

* 사진이나 동영상으로 가고 싶은 가게와 먹고 싶은 음식을 찾을 수있는 서비스.
* 영수증이나 리뷰를 게시하여 자신만의 음식·외출 로그를 남길 수 있
* 환경 : Xcode, Swift 상 최신화 지향, Combine, UIKit, SwiftUI
* GraphQL, 이미지 및 동영상 처리, Google Map, Mapbox SDK 경험자 우대
* 사용 프로젝트 디자인 패턴 : RIBs
* 업무도 : Github Ent, Confluence, Slack
* [https://apps.apple.com/jp/app/id1444842042](https://apps.apple.com/jp/app/id1444842042)
