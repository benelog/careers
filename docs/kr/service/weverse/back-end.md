# Back-end

서비스를 24시간 안정적으로 유지하기 위해 분산 처리, 비동기 처리, 모니터링 자동화 방법 등을 적용/운영하고 있습니다. 다양한 분야가 많기 때문에 특정 기술에 얽매이지 않고 적절한 상황에 맞는 기술(Java, Vertx, Redis, Kafka, Hadoop, Spark, Kubernetes 등)을 적용하고 있어 다양한 경험을 할 수 있고 누구나 새로운 기술을 제안하고 적용할 수 있는 기회가 열려 있습니다.

## 담당 업무

* Weverse 서버 플랫폼 개발
* 커뮤니티 기능 개발
* 라이브 및 동영상 기능 개발
* 서비스 관리 및 운영 툴 개발
* 대용량 데이터 처리 및 운영 업무
* 실물 응원봉 연동, 4K, 멀티뷰 등 다양한 서비스 업무 수행

## 업무 요건

* Java, Spring 개발 경험
* GIT 사용 경험 및 이해
* MSA 아키텍처 이해 및 RESTful API 설계
* Spring Framework 기반 웹서비스 개발
* 다양한 사람들과의 협업 능력
