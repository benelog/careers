# Front-end

## 담당 업무

* Weverse 서비스 개발&#x20;
* 라이브 동영상 서비스 개발&#x20;
* 스타와 팬을 연결시켜 주는 커뮤니티 플랫폼 개발&#x20;
* 글로벌 환경 최적화 (네트워크, 결제, 언어, 문화 등)

## 업무 요건

* 프론트엔드 개발 경력&#x20;
* HTML/CSS 등 마크업 관련 이해도를 바탕으로 업무에서 활용이 가능하신 분&#x20;
* Node.js, Typescript, react 등을 이용한 웹서비스 실무 경력&#x20;
* javascript framework 의 동작 원리 이해를 바탕으로 성능을 고려한 개발 경험/역량 있으신 분&#x20;
* 끈기있게 문제를 해결하고 적극적 태도로 업무에 참여하시는 분

## 우대 요건

* HTTP, REST, DNS, oAuth 등 인터넷 기반 프로토콜/기술에 대한 이해&#x20;
* HTML/CSS 등 마크업 관련 기본 지식&#x20;
* Javascript (최소 ES5 이상) 지식&#x20;
* Node.js, React, Redux, express 등 프레임워크 지식&#x20;
* 테스트, build, deploy 등 개발 및 운영 환경 구축 관련 지식&#x20;
* 글로벌 서비스 경험이 있으신 분&#x20;
* K-POP 아티스트 등 셀럽에 대해 관심이 많으신 분
