# Back-end

## 뉴스플랫폼 개발 

#### 이런 일을 맡고 있어요

뉴스플랫폼팀은 기사 데이터 입고, 정제, 가공 및 자동화된 큐레이션 등 뉴스 서비스 운영의 전과정에 필요한 다양한 플랫폼을 개발하고 있습니다.

대용량 트래픽 기반의 분산/비동기 처리 플랫폼, 미디어 비즈니스에 필요한 콘텐츠 편집 및 사이트 관리 도구, 콘텐츠 가공/분석 플랫폼 등 다양한 비즈니스 연관 서비스와 플랫폼들을 개발/운영하며 네이버 뉴스의 변화를 함께 만들어 갈 역량 있는 동료 개발자분들을 모십니다.

#### 입사하게 되면 담당할 업무입니다

* 스마트 콘텐츠 스튜디오(미디어 콘텐츠 큐레이션 애플리케이션) 개발
* 미디어 콘텐츠 파이프라인 개선 및 융합 콘텐츠 재생산
* 서비스 로그 공통 수집 및 분석 체계 통합
* MSA 기반의 서비스 지지를 위한 공통 API G/W 개발

#### 업무를 수행하기 위해 필요한 자격 요건입니다

* Java/Spring 혹은 Node.js 프레임워크 기반의 웹서비스 개발 경험
* RDBMS 기반의 데이터 모델링 및 개발 경험
* 객체지향, 리팩토링, 자동화 테스트에 대한 기본 지식
* 웹 사이트 및 API 운영, 개발 경험

#### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다

* MongoDB, Redis 기반의 응용 개발 혹은 성능 개선 경험
* 이벤트 기반 데이터 파이프라인 구축이나 MQ를 활용한 어플리케이션 개발 경험
* Restful, GraphQL API 구축 및 개발 경험
* Docker, Kubernetes 등 컨테이너 플랫폼에서의 빌드 및 운영 경험
* 다양한 템플릿 기반의 CMS 구축 및 반응형 사이트 개발 경험

## 프리미엄콘텐츠플랫폼 개발 

#### 이런 일을 맡고 있어요

프리미엄콘텐츠플랫폼팀은 **콘텐츠 창작자의 지속적이고 안정적인 콘텐츠 생산을 지원하기 위한 프리미엄콘텐츠라는 서비스**를 개발하는 업무를 진행하고 있습니다. 서비스 특성 상 네이버 내의 다양한 부서와의 협업을 통해 기본적인 백엔드 / 프론트엔드 개발을 비롯한 데이터 수집, 분석, 통계 또는 유료 결제와 관련된 영역 등 다양한 영역에서의 업무를 경험할 수 있습니다.

![](/careers/assets/kr/premium_area.PNG)

특히 대용량 트래픽 환경에서 서비스를 보다 빠르고 안정적으로 운용하기 위한 고성능/초고가용성 플랫폼을 구축/운영하며, 기술에 기반한 높은 수준의 서비스 운영 자동화와 관련 툴의 개발, 여러 백엔드 기술을 경험하며 디지털 콘텐츠 도메인의 전문가로 성장할 수 있습니다.

콘텐츠 창작, 관리,​유통. 분석 등의 전체 라이프 사이클을 경험 할 수 있으며, 보다 가치있는 콘텐츠의 발굴과 수익화를 지원하기 위한 비즈니스 인텔리전스까지 개발자가 주도 할 수 있는 다양한 실험 환경과 기술 과제들이 여러분들을 기다리고 있습니다.

#### 입사하게 되면 담당할 업무입니다

* 프리미엄콘텐츠 서비스 기반 플랫폼 고도화 (대용량 트래픽 / 비상대응 / 장애관제 시스템 개발)
* 프리미엄콘텐츠 서비스 개발
* 프리미엄콘텐츠 서비스에서 발생하는 다양한데이터를 유연하게 수집 / 분석 / 처리하는 플랫폼 개발
* 사용자 활동에 기반한 데이터 분석과 트렌드 및 추천 비즈니스 모델 개발

#### 업무를 수행하기 위해 필요한 자격 요건입니다

* Java / Spring Framework 또는 node.js에 익숙하신 분
* 서비스 요구조건을 빠르게 이해하고 데이터와 API를 디자인 할 수 있는 분
* 고가용성의 확장 가능한 시스템/설계 경험이 있으신 분
* 대용량 데이터 설계에 대한 기본 지식을 갖추신 분
* 웹 서버 개발과 유지보수, DB 모델링 및 시스템 설계 경험이 있으신 분

#### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다

* Restful API 이해와 고가용성의 API 구축 경험
* NoSQL 및 Message Queue 시스템 개발/운영 경험
* Spark 등 실시간 데이터 분석 및 파이프라인 처리 경험
* 대용량 데이터 배치 개발 경험
* 고성능/대용량 서버 개발, 대용량 데이터 경험
* 데이터 분석 및 통계 지식을 갖추신 분

## 스포츠플랫폼 개발 

#### 이런 일을 맡고 있어요

스포츠플랫폼은 스포츠 라이브, 영상, 기사 , 중계, 경기 기록 등 다양한 스포츠 컨텐츠를 기반으로 국내 프로 스포츠부터 해외 스포츠, 그리고 올림픽, 월드컵 같은 대형 이벤트까지 보다 빠르고 안정적인 서비스 운영을 목표로 네이버 스포츠 웹서비스를 제공하고 있습니다.

최근 보는 스포츠가 아닌 참여형 스포츠로의 변화를 준비하며 사용자 기반의 데이터를 위한 시스템 구축과 기존 서비스의 안정성을 높이기 위한 전반적인 리아키텍쳐링이 주요 목표입니다.

#### 입사하게 되면 담당할 업무입니다

스포츠 서비스개발자는 스포츠 서비스에서 필요한 플랫폼 및 전반적인 기능들을 신규 개발 및 운영하며 다양한 기획 운영 과제 및 개발 개선 과제를 수행합니다.

* 다양한 스포츠 종목의 데이터를 처리하는 수신 시스템 신규 개발
* MSA 기반의 백엔드 시스템 개발  및 React 기반의 UI 개발
* 사용자 참여 기반 스포츠 참여 서비스 신규 개발

#### 업무를 수행하기 위해 필요한 자격 요건입니다

* Spring Framework 기반의 웹어플리케이션 개발/운영 경험
* 서비스 요구조건을 빠르게 이해하고 데이터와 API를 디자인 할 수 있는 분
* 고가용성의 확장 가능한 시스템/설계 경험이 있으신 분
* 웹 서버 개발과 유지보수, DB 모델링 및 시스템 설계 경험이 있으신 분

#### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다

* 고성능/대용량 서버 개발, 대용량 데이터 경험
* Docker 및 Kubernetes 기반의 애플리케이션 관리 및 운영 경험
* React 기반의 웹개발 경험
* 스포츠를 좋아하시는 분

## 콘텐츠피드백플랫폼 개발 

#### 이런 일을 맡고 있어요

콘텐츠피드백플랫폼은 유저들의 피드백(공통댓글, 좋아요 등)을 안정적으로 처리할 수 있는 네이버 공통 플랫폼 개발을 하고 있으며, 네이버의 다양한 서비스에 적용되어 있습니다. 공통 플랫폼 특성상 서비스에서 빠르고 쉽게 적용이 가능해야하기 때문에 백엔드 및 프론트엔드 개발을 통해 SDK/API를 제공하고 있으며, 다양한 데이터의 실시간 수집, 분석, 통계, 자동화된 콘솔 개발등의 업무를 하고 있습니다.

댓글, 좋아요는 네이버의 다양한 서비스에 적용되어 있으며 이를 사용하는 유저들도 또한 많기 때문에 대용량 트래픽 처리 및 지속적으로 늘어나는 데이터의 안정성을 확보를 위한 초고가용성 플랫폼의 운영 및 개발을 경험할 수 있습니다.

* 공통댓글 플랫폼
  * 네이버 뉴스, 웹툰, vlive 등  50여개 이상의 서비스에 댓글 및 채팅을 플랫폼으로 제공
* 좋아요 플랫폼
  * 네이버 뉴스, 스포츠, 웹툰, 블로그, 카페 등  70여개 이상 서비스에 플랫폼으로 제공

#### 입사하게 되면 담당할 업무입니다

* 대량의 트래픽 대응 및 실시간 분석을 위한 MSA+EDA 기반 플랫폼 개선
* 지속적으로 증가하는 데이터의 안정적인 처리를 위한 저장계 개선
* 공통댓글 및 좋아요 플랫폼 자동화를 위한 콘솔 개발
* 공통댓글, 좋아요 플랫폼에서 발생하는 다양한 데이터를 빠르게 수집 / 분석 / 처리하는 데이터 분석 개발

#### 업무를 수행하기 위해 필요한 자격 요건입니다&#x20;

* Java, Spring Framework 에 익숙하신분
* RDBMS(MySQL/MSSQL/Oracle 등) 모델링 및 사용 경험
* Redis 기반의 응용 개발 혹은 성능 개선 경험
* 객체지향, 리팩토링, 자동화 테스트에 대한 기본 지식

#### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다

* 고성능/초고가용성 서비스/플랫폼 개발 및 운영 경험
* Restful API 이해와 고가용성의 API 구축 경험
* Reactive Streams 에 대한 기본적인 이해 및 관련 구현체(Reactor) 사용 경험
* MongoDB, Hbase 등 NoSQL 기반의 응용 개발 및 성능 개선 경험
* Message Queue 시스템 개발/운영 경험
* Spark, flink 등 실시간 데이터 분석 및 파이프라인 처리 경험
* kubernetes 기반의 응용 프로그램 개발 경험
* 대용량 데이터의 batch 처리
