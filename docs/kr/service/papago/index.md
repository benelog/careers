# Papago

파파고는 기계번역, 음성인식, 음성합성, 문자인식 등의 기술이 집약된 통번역 서비스입니다. 언어의 장벽이 없는 세상을 만들기 위해 같이 고민하고 성장해 나갈 개발자를 찾고 있습니다.

새로운 시도와 도전을 하고 싶으신 분, 서비스와 함께 성장하실 분을 기다립니다.
