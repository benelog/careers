# AI/ML

## Machine Learning : Computer Vision (CV)

#### 이런 일을 맡고 있어요

파파고는 2016년부터 기계번역 서비스를 시행한 조직으로, 단순히 연구를 하는데 그치지 않고 우리가 만든 기술을 실제 서비스에 적용해서 사용자의 만족도를 높이는 경험을 할 수 있습니다.\
파파고의 CV 머신러닝 과학자들은 파파고 이미지 번역팀에 속해 이미지 번역에 필요한 기술을 연구/개발합니다.\
보통 다음과 같은 문제를 해결합니다.

* Object Classification / Detection / Segmentation / Tracking
* Text Detection / Recognition
* Document Analysis and Recognition
* Text Inpainting / Editing
* Image Enhancement / De-warping
* Model Compression

#### 입사하게 되면 담당할 업무입니다

* 파파고의 이미지 번역 성능을 높이기 위한 문제점 확인 및 분석 (모델의 성능 측정)
* 이미지 번역의 성능과 사용 편의성을 높이기 위한 방법 연구 (데이터 수집, 정제 / 모델의 고도화 / 경량화, 최적화)

#### 업무를 수행하기 위해 필요한 자격 요건입니다

* 스스로 문제를 정의하고 해결하기 위한 모델링이 가능한 분
* 최신 머신러닝 알고리즘과 관련 기술에 대한 이해를 가진 분
* Python 등의 프로그래밍 언어를 다룰 수 있는 분
* Tensorflow, Pytorch, MXNet 등의 딥러닝 프레임워크를 1개 이상 다룰 수 있는 분

#### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다

* 최신 머신러닝 알고리즘과 관련 기술에 대한 이해를 가진 분(관련 연구/개발 경력 2년 이상)
* Kaggle 등 머신러닝 관련 대회에서 좋은 성과를 거두거나 국제 학술 대회에 논문을 게재하신 분

## Machine Learning : Natural Language Processing (NLP)

#### 이런 일을 맡고 있어요

파파고는 2016년부터 기계번역 서비스를 시행한 조직으로, 단순히 연구를 하는데 그치지 않고 우리가 만든 기술을 실제 서비스에 적용해서 사용자의 만족도를 높이는 경험을 할 수 있습니다.\
파파고의 NLP 머신러닝 과학자들은 파파고 기계번역팀, 이미지번역팀에 속해, 번역 모델은 물론 머신러닝 기반의 다양한 자연어처리 모델을 만듭니다.\
보통 다음과 같은 문제를 해결합니다.

* Neural Machine Translation
* Quality Estimation
* Multilingual NLP
* Sentence Segmentation
* Language Detection
* Text Recognition / Classification / Correction / Analysis

#### 입사하게 되면 담당할 업무입니다

* 자연어처리 기술을 빠르게 프로토타이핑하고 학습
* 자연어처리 모델이 서비스 상에서 대용량의 요청을 빠르게 처리할 수 있도록 최적화
* 자연어를 과학적으로 탐구하고 심층적으로 연구

#### 업무를 수행하기 위해 필요한 자격 요건입니다

* 스스로 문제를 정의하고 해결하기 위한 모델링이 가능한 분
* 최신 머신러닝 알고리즘과 관련 기술에 대한 이해를 가진 분
* 주요 머신러닝 프레임워크에 대한 이해가 깊은 분 (ex: TensorFlow, PyTorch)
* Python, C++ 등의 프로그래밍 언어를 1개 이상 자유자재로 다루고 Linux/Shell 환경에 익숙하신 분
* 영문 논문을 빠르게 읽고 이해하는 데 어려움이 없으신 분
* 뛰어난 의사소통 능력을 가진 분

#### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다

* 자연어 텍스트 또는 음성 데이터에 대한 경험이 있으신 분
* ML/NLP 관련 상용 프로덕트를 개발한 경험을 가진 분
* Kaggle 등 머신러닝 관련 대회에서 좋은 성과를 거두거나 국제학술대회에 논문을 게재하신 분
