# Back-end

Papago는 글로벌 번역 서비스 파파고와 사전을 서비스하는 조직입니다.  
Papago Server 팀은 파파고 번역 및 사전 서비스 전반에 대한 software engineering이 주요 업무입니다.  
세부적으로는 DB모델링, API 개발, 로깅 빅데이터 분석, 분산처리, 모듈 serving, performance tuning, containerization/orchestration, continuous integration 등을 경험할 수 있습니다.

수평적인 분위기 안에서 경쟁보다는 서로 협력하며 장기적인 목표를 가지고 함께 성장해 나가는 팀입니다.  
팀 내에 다양한 스터디가 항상 열려있고, 스스로 원하는 스터디를 개설할 수도 있습니다.  
직군 구분 없이 좋은 아이디어가 있다면 내 아이디어로 서비스를 만들어갈 수 있습니다.

새로운 서비스를 계속적으로 만들어 나가는 조직으로 legacy보다는 최신 트렌드로 새로운 프로젝트의 설계부터 구현까지 좋은 경험의 기회가 열려있습니다. 또한 대용량 트래픽 운영을 경험할 수 있고, GPU 서버를 접할 수 있으며 여러 나라 동료들과 협업해 볼 수 있다는 측면에서 일반적인 BE팀과 차별화된 Career Path를 가져갈 수 있다 생각됩니다.

## Papago Translate 서비스 Back-end 개발

#### 입사하게 되면 담당할 업무입니다.

* Spring 기반 Papago Translate 서비스 Back-end 프로젝트 개발 및 운영

#### 업무를 수행하기 위해 필요한 자격 요건입니다.
