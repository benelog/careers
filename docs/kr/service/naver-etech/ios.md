# iOS

## 이런 일을 맡고 있어요. 

### PRISM 모바일 라이브 방송 및 그래픽스 기술 개발

* 차별화된 "모바일 라이브 송출 및 비디오 편집" 기술기반으로 SDK 개발하여 네이버 전사 동영상 서비스를 지원.
  * 라이브 스트리밍 SDK : 쇼핑 라이브, V LIVE, PRISM LIVE STUDIO 등
  * VOD 편집 SDK : Moment, 지식iN, 네이버 포스트, 카페 등
  * 안정적인 영상 기술 기반으로, 혁신 기술을 신속하게 리서치 / 상용화하여 네이버 서비스들에 지원 
  * 안정적인 모바일 네트워크 송출 기술 / 영상처리 기술의  연구와 상용화를 병행

### PRISM Live Studio 앱 개발

* PRISM Live Studio 모바일 앱을 개발 및 운영
* PRISM 앱을 통해 누구나 쉽게 라이브 방송 기능과 동영상 편집 기능으로 동영상 콘텐츠를 생산할 수 있도록 지원
* 200여 국가 / 해외 사용자 비율이 70% 에 이르는 글로벌 서비스로써, 다양한 사용자와 소통 / 기능 개발 

### SmartEditor 모바일 개발

* 네이버의 다양한 글을 생산하는 핵심 도구의 SDK기능을 개발 / 운영
* 문서 플랫폼으로서의 진화를 목표로 도전
* UGC 문서 쉽게 작성하기 위한 편집기의 기본적인 인터랙션과 적용이 간편한 플랫폼 구조 설계
* 글로벌 서비스 환경 / 다국어지원 환경에서 기능 개발 및 운
