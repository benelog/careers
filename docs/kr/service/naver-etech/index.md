# ETECH

## ETECH를 소개합니다. 

**NAVER E**ntertainment & **E**merging **TECH**nology 는  
포토 / 오디오 / 비디오 &lt; 생산 - 클라우드 - 소비 &gt; 전체 사이클의 기술을 연구 개발하는 플랫폼 그룹입니다.

Beyond Screen, Being There !  
공연장 VIP석에서 느끼는 그 생생함, 감동 그리고 소통을 안방 1열로 전달하기 위한 미션을 바탕으로  
현실감 극대화를 위한 Immersive 미디어, 글로벌 스트리밍 클라우드, AI 그리고 XR 프로젝트들을 진행하고 있습니다.

## 우리는 이런 개발 업무를 하고 있어요.

### 동영상 플랫폼

ETECH 동영상 플랫폼은 영상 처리 기술을 기반으로,  
**LIVE & VOD 클라우드 / 통합 플레이어 / PRISM Mobile SDK / PRISM LIVE Studio APP**을 개발, 제공합니다. 

![](/careers/assets/kr/2021-08-19-17.45.37.png)



### 오디오 플랫폼

ETECH 오디오 플랫폼은 오디오 처리 기술을 기반으로,  
**오디오 클라우드 / 통합 플레이어**를 개발, 제공합니다. 

![](/careers/assets/kr/2021-08-19-17.51.11.png)

### 포토 플랫폼

ETECH 포토 플랫폼은 이미지 처리 기술을 기반으로   
**포토 클라우드 / 포토 모바일 SDK**를 개발, 제공합니다.

![](/careers/assets/kr/2021-08-19-18.05.07.png)

### UGC 제작 플랫폼

#### [SmartEditor](https://smarteditor.naver.com)

네이버의 다양한 서비스에서 콘텐츠를 생산 / 소비하는데 이용하는 UGC 도구로써,  
네이버의 콘텐츠를 풍성하게 만들어, 서로의 경험과 지식을 나누는 허브 기능을 개발, 제공합니다.

#### Home Builder

홈빌더는 네이버 서비스에 자신의 명함과 같은 홈을 만드는 WYSIWYG 플랫폼입니다.
