홈빌더는 웹 안에 있는 수많은 정보와 기능을 담을 수 있는 그릇입니다. 홈빌더를 이용하면 홈을 더 다양하게 만들 수 있고, 다양한 홈은 새로운 비즈니스 기회를 창출합니다.

* UGC 홈은 새로운 작가의 등용문이 되고,
* 쇼핑 홈은 상품의 판로를 열고,
* 기업의 홈은 사업을 홍보하는 통로가 됩니다.

여기에서 더 나아가 플랫폼 생태계와 비즈니스 영역까지 확장할 수 있습니다.

* 잘 만든 사용자의 컴포넌트와 테마를 거래하는 마켓 플레이스
* 홈을 만들 수 있는 공간 임대 SaaS
* Shopify와 같은 SME 비즈니스 확장

홈빌더를 이용하면 이런 기회를 더 쉽고 빠르게 만들 수 있습니다.

### “홈”은 사용자의 개성을 표현하는 중요한 수단입니다. <a id="NAVER&#xAE30;&#xC220;&#xC9C1;&#xBB34;&#xC548;&#xB0B4;(HomeBuilder)-&#x201C;&#xD648;&#x201D;&#xC740;&#xC0AC;&#xC6A9;&#xC790;&#xC758;&#xAC1C;&#xC131;&#xC744;&#xD45C;&#xD604;&#xD558;&#xB294;&#xC911;&#xC694;&#xD55C;&#xC218;&#xB2E8;&#xC785;&#xB2C8;&#xB2E4;."></a>

우리는 온라인 서비스로 다른 사람과 나의 일상과 정보를 공유합니다. 온라인 세상 사용자에게 자신의 정체성은 중요한 의미를 가집니다. 온라인에서 '홈' 은 사용자의 개성을 표현하는 수단입니다.

* 공공기관의 홈은 유용한 소식을 시민들에게 알리는 게시판이며
* UGC의 홈은 나의 소식을 다른 이들과 공유하는 공간이 되고
* 쇼핑몰의 홈은 중소 상인들이 물건을 파는 판매대로 사용됩니다

홈빌더는 사용자가 다양한 레이아웃과 데이터를 조합하여 자신의 개성을 마음껏 표현하는 홈을 만들 수 있게 돕습니다.

* 사용자가 코드 없이 UI로 미려한 홈을 만들 수 있는 도구 제공
