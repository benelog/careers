# HomeBuilder 개발

## 이런 일을 맡고 있어요 <a id="id-(JD)HomeBuilder&#xAC1C;&#xBC1C;-&#xC774;&#xB7F0;&#xC77C;&#xC744;&#xB9E1;&#xACE0;&#xC788;&#xC5B4;&#xC694;"></a>

홈빌더는 다양한 네이버 서비스에 홈을 제공합니다. 여러 서비스에 적용하기 위해 기능 모듈의 플랫폼 구조로 개발하고 있습니다.

* 홈을 편집하는 SPA 방식의 wizwig 모듈화 Editor
* 편집한 홈을 그려주는 npm module Renderer
* 홈 End 화면의 동적인 처리를 위한 Viewer JS
* NodeJS를 사용한 대용량 트래픽 처리 BackEnd API

이러한 플랫폼  Stack 을 Javascript 언어로 구현합니다.

모듈화를 위해  Editor / Renderer를  컴포넌트 방식으로 설계 개발하고, 서비스에서 자유롭게 커스터마이징 하도록 제공하며, 홈을 구성하는 외부 데이터를 정제하고 이를 처리 하기 위한 대용량 트래픽의 분산 처리를 구현합니다.

홈빌더는 영역을 나누지 않고 모든 구성원이 Full Stack 기반으로 같이 고민하고 개발하고 있습니다.

## 우리 같이 해봐요 <a id="id-(JD)HomeBuilder&#xAC1C;&#xBC1C;-&#xC6B0;&#xB9AC;&#xAC19;&#xC774;&#xD574;&#xBD10;&#xC694;"></a>

* 홈빌더 플랫폼 Front-End 개발
* 홈빌더 플랫폼 Back-End API 개발
* 홈빌더 플랫폼 운영도구 개발
* 홈빌더 플랫폼 인프라 구축 / 운영
* 로깅 및 통계 개발

## 우리는 이렇게 일해요 <a id="id-(JD)HomeBuilder&#xAC1C;&#xBC1C;-&#xC6B0;&#xB9AC;&#xB294;&#xC774;&#xB807;&#xAC8C;&#xC77C;&#xD574;&#xC694;"></a>

### 기술 스택 <a id="id-(JD)HomeBuilder&#xAC1C;&#xBC1C;-&#xAE30;&#xC220;&#xC2A4;&#xD0DD;"></a>

* React 기반의 컴포넌트로 모듈화된 WIZIWIG Editor와 Renderer를 개발합니다.
  * TypeScript
  * React / Redux 
  * Lerna\(for monorepo\)
  * Webpack
  * ESLint, Jest
* 여러 환경의 성능 최적화를 통해 빠른 Application 속도를 보장합니다
  * 클라이언트 성능 최적화를 위한 브라우저 로딩/렌더링 최적화 작업
  * 반응형 웹
* 다양한 환경의 서비스 대응과 API 개발을 NodeJS 기반으로 개발하고, 네이버 인프라를 통해 배포 운영합니다.
  * N2C \(docker, kubernetes\)
  * NodeJS, Express, MongoDB
  * Nginx, Redis 
* 대용량 트래픽 장애 방지, 가용성 증대, 확장성 확보를 위한 엔지니어링 활동을 합니다.
  * 아키텍처 개선
  * 트래픽 제어 
  * CI / CD

### 개발 문화 <a id="id-(JD)HomeBuilder&#xAC1C;&#xBC1C;-&#xAC1C;&#xBC1C;&#xBB38;&#xD654;"></a>

* 2주 단위 스프린트를 기본 프로세스로 운영합니다.
* 이슈 티켓은 Jira로, 문서는 Confluence로 관리합니다.
* ESLint 같은 도구를 적극적으로 활용해 일관성 있는 코드 작성을 지향합니다.
* 코드 리뷰와 테스트 자동화 등의 건강한 개발 문화를 만듭니다.
* 회고를 통해 개발 프로세스를 점진적으로 개선합니다.
