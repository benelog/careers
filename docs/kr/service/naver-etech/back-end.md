# Back-end

## 이런 일을 맡고 있어요. 

### 이미지 처리 서버 개발

* 네이버 및 계열사에서 사용하고 있는 이미지 처리 / 변환을 위한 Server Side/Client Side 엔진 개발을 담당
* 대용량 / 고속 이미지처리 기술을 기반으로 AI 를 이용한 이미지 분석 및 처리 기술로의 확장 진행중

### 미디어 서버 개발

* NAVER 전사 동영상 서비스를 지원하는 LIVE / VOD 미디어 스트리밍 서버 개발
* 글로벌 환경에서 365 / 7 / 24 무장애로 지원하기 위한 
  * 안정성을 최대 목표
  * 효율성, 기술 경쟁력 강화를 위한 차별화된 선행기술 연구 및 개발 주요 미션
  * 안정적이고 효율적인 미디어 처리 기술과 BE 소프트웨어를 만들기 위한 개발 연구

### LIVE / VOD 클라우드 개발

* 네이버 전사 및 외부 서비스에 VOD 와 LIVE를 제공하는 글로벌 플랫폼 개발 / 운영
  * 동영상의 업로드, 변환, 재생까지의 모든 단계를 지원하는 플랫폼을 개발 
    * CloudNative \(Kubernetes 기반\) 로의 전환 개발
    * 대규모 동영상의 변환 / 재생을 위한 플랫폼 개발
  * 무장애 LIVE 방송과 새로운 LIVE 환경을 제공하는 플랫폼을 개발
    * 무중단 서비스와 고가용성을 유지하기 위한 기술 고도화와 LIVE 서비스 기능 개발
    * 실시간 대용량 트래픽 제어의 기술 고도화

### 포토 클라우드 개발 

* 네이버서비스에 노출되는 모든 이미지의 저장, 변환, 노출까지의 전 구간을 지원하는 플랫폼 개발 / 운영
  *  대용량 / 실시간 이미지 프로세싱 개발
  *  빅데이터를 다루는 클라우드기반 Enterprise Application을 개발
    * 하루 수십억 건의 이미지를 빠르고 안정적으로 서비스 할 수 있는 서버 플랫폼을 구축/운영
    * Cloud Native/MSA로의 전환 개발\(Kubernetes, ELK, golang, Kafka\)
    * 대용량 로그 데이터의 분석과 통계 시스템 개비
    * 대량으로 생성/소비 되는 이미지의 효율적인 호스팅, 멀티 Region 지원

### SmartEditor 서버 개발

* 네이버서비스에 노출되는 문서의 생성 / 변환 / 뷰어까지의 전 구간을 지원하는 플랫폼 개발 / 운영 
  * 최적화된 문서등의 UGC 데이터 구조 / 딜리버리를 위한 설계
  * UGC 데이터 저장 &lt;-&gt; 서비스와의 안정적인 WAS 연동 / 개발 / 운영
  * 글로벌 서비스를 위한 멀티리전 아키텍처의 설계와 구축
