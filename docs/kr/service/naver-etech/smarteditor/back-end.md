# Back-end

## 이런 일을 맡고 있어요

SmartEditor ONE 플랫폼\(이하 SE ONE 플랫폼\)은 NAVER 전사의 콘텐츠 저작 플랫폼으로서 다양한 서비스에 콘텐츠 생산/소비 환경을 제공하고 있으며, 월 단위 약 6,000,000건의 콘텐츠가 SE ONE 플랫폼을 통해 만들어지고 있습니다. Back-end 개발자는 SE ONE 플랫폼의 다양한 API를 개발합니다. 보다 나은 안정적인 성능을 위해 SRE\(사이트 신뢰성 엔지니어링\), 대규모 문서 데이터 기반 통계 시스템을 개발합니다. 마지막으로 UX/UI 설계와 Front-end 개발자, iOS 개발자, Android 개발자, 테스트 엔지니어와 함께 고민하고 제품을 만들어 갑니다.

## 우리 같이 해봐요

* SE ONE 플랫폼 인프라 구축/유지보수 및 애플리케이션 개발/운영
* SE ONE 플랫폼 글로벌 서비스를 위한 멀티 리전  아키텍처 설계 및 개발/운영
* SE ONE 플랫폼 신규 기능 RESTful API 개발
* SE ONE 플랫폼 문서 모델 설계 및 스키마 관리
* 대규모 문서 데이터 기반 문서 통계 시스템 개발
* AI 기반 신규 기능 개발
* SRE
* 다양한 로그 수집 및 분석

## 우리는 이렇게 일해요

* 최적화된 사용자 경험의 멀티플랫폼 WYSIWYG 편집기를 만들기 위해 UX/UI 설계와 Front-end 개발자, iOS 개발자, Android 개발자와 긴밀하게 협업합니다.
* 대용량, 클라우딩 컴퓨팅 환경에 맞는 Java 기반 MSA를 개발합니다.
  * Java
  * MySQL
  * MongoDB
  * Kafka
  * ELK
* Spring 프레임워크 이해를 바탕으로 글쓰기에 필요한 API 및 라이브러리를 개발합니다.
* 네이버와 글로벌 서비스의 장애 예방을 위해 고민하고 해결합니다. \(SRE 활동\)
  * n2c \(docker, kubernetes\)
  * 아키텍처 구조 개선
  * 트래픽 제어
  * CI/CD
  * 모니터링, 테스트 강화
* 사용자 글쓰기 지원을 위해 AI 기반 문서 생산도구를 개발합니다.
  * 컴포넌트 어시스턴트 \(GPT-3\)
* 문서 통계 및 분석을 위해 데이터 파이프라인, 데이터 레이크를 구축하고 운영합니다.
  * Kafka
  * c3s \(hadoop ecosystem\)
  * Spark
  * Zeplin

## 이런 기회가 있습니다.

* 네이버와 글로벌 서비스의 대용량 데이터 처리 인프라 설계부터 개발까지 경험할 수 있습니다.
* 네이버와 글로벌 서비스의 사용자를 대상으로 하는 기능을 설계하고 개발할 기회가 있습니다.
* API 개발, SRE 활동, 데이터 엔지니어링, AI 기반 제품 개발 등 다양한 분야를 골고루 경험할 수 있습니다.

## 이미 당신은 준비되어 있네요

* Java의 이해 및 개발 역량 보유
* Spring 프레임워크의 구조와 매커니즘의 이해
* 실 서비스 중인 애플리케이션 개발 및 운영 경험
* Linux 환경의 인프라 운영 능력
* 웹 기본 미들웨어\(Apache, NGINX, Tomcat\) 활용 능력
* 다양한 유관 서비스와의 협업 능력

## 굳이 없어도 되지만, 있으면 사양하지 않아요

* 대용량 트래픽 서비스 경험
* Docker, Kubernetes를 사용한 서비스 개발 및 운영 경험
* 대형 시스템의 분산 환경 RDB 또는 NoSQL\(MongoDB, ES 등\) 사용 경험
* UGC\(User Generated Content\) 기반 서비스 이슈 대응 경험
* Kafka 클러스터에 대한 이해와 개발 역량 보유
* Spark, Spark 스트림을 활용한 개발 역량 보유
* 지표 데이터 수집, 가공, 처리, 분석 경험
