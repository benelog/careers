# ETECH

## ETECH를 소개합니다.  

**NAVER E**merging **TECH**nology 는\
포토 / 오디오 / 비디오 < 생산 - 클라우드 - 소비 > 전체 사이클의 기술을\
연구 개발하는 멀티미디어 플랫폼 그룹입니다.

<figure><img src="../../.gitbook/assets/스크린샷 2023-01-12 오후 2.03.39.png" alt=""><figcaption></figcaption></figure>

ETECH는 **"글로벌 Top-Tier의 미디어 기술 경쟁력으로 서비스의 성장을 지원하고 견인한다."** 의 미션으로

* 공연장 VIP석에서 느끼는 현장감과 감동, 그리고 소통을 안방 1열로 전달하는 공연 컨텐츠로.
* 장소 구애없이, 라이브 강의를 통한 온라인 교육으로.
* 사람과 디지털휴먼이 함께 참여하는 직관적이고 입체적인 상품 소개와 구매 경험을 위한 라이브 커머스 쇼핑 문화로.

시간 / 공간 / 용량의 제약 사항을 극복하고, 생생한 현장 느낌과 안정적인 지원을 위한\
Immersive 미디어, 글로벌 스트리밍 클라우드, AI 그리고 XR 프로젝트의\
이머징 기술연구와 개발을 통해 글로벌 미디어 시대를 준비하고 있습니다.

## 우리는 이런 개발 업무를 하고 있어요.

### 동영상 플랫폼 

ETECH 동영상 플랫폼은 영상 처리 기술을 기반으로,\
대용량 트래픽을 안정적으로 대응하기 위한 엔터프라이즈 플랫폼과 \
동영상 컨텐츠 생산과 소비를 활성화하는 기반을 조성하여 네이버의 동영상 생태계를 만들어갑니다.&#x20;

* LIVE 클라우드
* VOD 클라우드
* PRISM 플레이어 ( Android / iOS / Web )
* PRISM Mobile SDK ( Android / iOS )
* PRISM LIVE Studio APP ( Android / iOS / Mac / Windows )

![](/careers/assets/kr/%EC%8A%A4%ED%81%AC%EB%A6%B0%EC%83%B7%202021-08-19%2017.45.37.png)



### 오디오 플랫폼 

ETECH 오디오 플랫폼은 오디오 처리 기술을 기반으로, \
글로벌 환경에서 대용량 트래픽을 안정적으로 서비스하기 위한 플랫폼을 지원합니다.

* AOD 클라우드
* PRISM 플레이어 ( Android / iOS / Web )

![](/careers/assets/kr/%EC%8A%A4%ED%81%AC%EB%A6%B0%EC%83%B7%202021-08-19%2017.51.11.png)

### 포토 플랫폼 

ETECH 포토 플랫폼은 이미지 처리 기술 기반과,\
Real-time 처리와 대용량 트래픽을 안정적으로 수용하는 플랫폼 기술로\
네이버 서비스의 이미지 생산과 소비를 담당하고 있습니다.

* 포토 클라우드
* 포토 Mobile SDK ( Android / iOS )&#x20;

![](/careers/assets/kr/%EC%8A%A4%ED%81%AC%EB%A6%B0%EC%83%B7%202021-08-19%2018.05.07.png)
