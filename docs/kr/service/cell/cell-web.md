# 밴드 Cell Web

## Who We Are (조직 소개)

* 우리는 이런 일을 합니다.
  * 한국을 넘어 전세계를 향해 가고 있는 글로벌 커뮤니티 서비스 BAND의 웹 Front-End 기능 개선 및 유지 보수 업무를 하고 있습니다.
  * Legacy 환경에서 벗어나 React 환경 기반으로 새로 태어날 준비를 하고 있습니다. <br>
    (이미 팀내에서는 Angular, Vue, React 환경에서의 다수의 프로젝트 개발 경험을 가지고 있음)
* 우리는 아래와 같은 팀을 지향합니다.
  * 새로운 도전과 변화에 항상 열려있는 팀
  * 개개인의 실험 정신을 맘껏 발휘 할 수 있는 팀
  * 상대방의 의견에 항상 열려있고 긍정적으로 서로 소통하는 팀
  * 팀의 성장 노력 속에서 성장하고 있는 나를 발견할 수 있는 팀
  * 일하는 재미가 있는 팀

## What You'll Do (담당 업무)

* 네이버의 글로벌 커뮤니티 서비스인 BAND 서비스의 글로벌 최적화 및 개선 업무
* Legacy Framework 기반의 BAND 서비스 유지 보수
* React 기반으로 BAND의 PC WEB 및 Mobile WEB 전면 재구축 진행 업무

## Required Skill (필수 요건)

* 웹 서비스 및 HTML, CSS, Javascript 에 대한 기본 지식 보유
* modern framework 이전 legacy framework 환경에서 구현 경험이 있으신 분
* React 환경에서 구현 경험이 3년 이상 보유하신 분
* 1가지 이상의 React 상태 관리 lib 사용 경험이 있으신 분
* RESTful API에 대한 이해 및 활용 지식 보유
* 변화와 발전, 타협에 열린 자세

## Preferred Skill (우대 요건)

* React, Vue, Angular 등의 최신 Modern Framework 기반하에서 경험이 풍부하신분
* Webpack/vite등 다양한 빌드 도구를 통해 시스템 구성 및 최적화에 경험이 많으신 분
* FrontEnd뿐만 아니라 BackEnd에 대한 경험 및 기본 지식을 갖추신 분
* FrontEnd 개발 방법론에도 관심이 많아 항상 최적화에 대한 고민을 하시는 분
* 디자인 시스템 적용 및 운용 경험이 풍부하신 분
* 글로벌 서비스 개발 경험이 있으신 분
* FrontEnd 에서의 테스트 코드 경험 및 관심이 많으신 분
* Sentry 운용 및 적용 경험이 있으신 분
* Legacy Framework 에서 Modern Framework로 전환 경험이 있으신 분.
