# Music

Tune CIC는 음악(Tune)과 채널 고정(Stay Tuned)이라는 의미를 담고 있으며, 미세한 음의 차이까지 튜닝하듯, ​사용자와 시장의 니즈에 민감하게 반응하고 ​감각적으로 깨어 있는 서비스를 만들기 위해 최선의 노력을 기울이고 있습니다.&#x20;

파티룸, 노래방, 아티스트쇼, 뉴오디오 등 이 있습니다.

Music Dev는

바이브, 오디오클립, 라인뮤직 Android 앱을 개발 담당하는 [Music Android](music-android.md) /[ LINE MUSIC Android 팀](line-music-android.md)

바이브, 오디오클립, 라인뮤직 iOS 앱을 개발 담당하는 [Music iOS](music-ios.md) / [LINE MUSIC iOS 팀](line-music-ios.md)

바이브 서비스, 오디오클립 서비스 서버 개발을 담당하는 [Music Server 팀  ](music-server.md)

뮤직 서비스 전반의 데이터를 쉽게 저장, 활용할 수 있는 고성능/대용량 플랫폼 서버 어플레이케이션과 파티룸 서버 개발을 담당하는 [Music Platform 팀](music-platform/)

일본 라인뮤직 서버 개발을 담당하는 [LINE MUSIC Server 팀](line-music-server.md)

바이브, 라인뮤직 추천 기능과 데이터 인사이트를 제공하고 개발하는 [Music AI & Data Tech 팀](music-ai.md)

바이브, 라인뮤직 서비스 품질 관리 및 QA를 담당하는 [Music Quality Engineering 팀](music-quality-engineering.md)

으로 이루어져  고객들에게 재미있고 즐거운 다양한 기능을 제공하기 위해 열심히 재미있게 개발하고 있습니다.&#x20;
