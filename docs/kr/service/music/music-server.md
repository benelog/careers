### #다양한 기술





### **Back-end 개발조직으로 VIBE, 뉴오디오, 오디오클립 서비스의 심장인 서버 개발을 담당하고 있습니다.**

* 대규모/대용량 DB 운영환경을 경험할 수 있으며, 대규모 분산처리 및 모듈화를 경험할 수 있습니다.
* 추천, 개인화 기반 기술을 경험할 수 있습니다.
* 스트리밍 서비스 개발 신기술을 진보적으로 활용해 보실 수 있습니다.
* 새로운 서비스 개발에 적극적으로 참여 하실 수 있습니다.
* 베트남, 중국, 대만, 일본 등의 해외에서 기술협업을 경험하실 수 있습니다.



### **업무를 수행하기 위해 필요한 자격 요건입니다.**

* Java/Spring 기반의 애플리케이션 서버 개발 경험이 있으신 분&#x20;
* Oracle 또는 MySQL과 같은 RDBMS 경험이 있으신 분&#x20;
* 새로운 기술에 대한 호기심이 많고 적극적으로 습득하고자 하시는 분&#x20;
* 음악을 좋아하고 스트리밍 음악 서비스를 즐겨 사용하시는 분



### 추가로 보유하고 있다면 좋을 스킬셋이나 경험입니다·&#x20;

* Kafka, ES, 하둡 개발 및 대용량 데이터 처리 개발 경험이 있으신 분.&#x20;
* 데이터 분산처리 경험이 있으신 분.&#x20;
* MSA(Microservice Architecture), container orchestration(docker, K8S)기반의 컨테이너화된 애플리케이션 개발 경험이 있으신 분&#x20;
* JPA, Hibernate 등 ORM 사용과 DDD(Domain Driven Design) 구조의 개발 경험이 있으신 분&#x20;
* MongoDB 기반의 도큐먼트 지향적인 스키마 설계가 가능하신 분



### 참고로 아래와 같은 기술을 사용하고 있습니다. 이에 대한 경험이 있으시면 더 좋습니다.

* Languages : Java, Javascript, Shell script
* Frameworks : Spring, Spring Boot
* Data Sources : ORACLE, MySql, MongoDB, Hadoop(Hive, HBase, Spark), Redis, Elastic Search, Kafka
* Development Tools : Intellij, Maven, Gradle, Jenkins, Git
* FE 개발은 하지않습니다.
