# OPEN-API

### 'VIBE-\*\*\*' 과 연동하여 다양한 콘텐츠 제공

**OPEN-API** (Kotlin + Spring Cloud + K8S) 플랫폼은 **VIBE** 서비스를 **사내**(블로그, 검색..) 또는 협의된 **사외**(음향기기, 카플레이어, 게임..) 서비스에서 손쉽게 앨범/트랙/아티스트/믹스테잎/플레이리스트/보관함/차트/검색/재생과 같은 VIBE 서비스를 자체적으로 커스텀하게 구축 할 수 있도록 웹 기반의 표준화된 인증 방식(네이버 로그인 OAuth 2.0)과 프로토콜(GraphQL) API를 제공하는 플랫폼입니다.

![OPEN-API Architecture](/careers/assets/kr/music-open-api.png)

연동하고자 하는 시스템에서는 **Vibe Developer Center** 통해서 **Web API**, **Web SDK**(Playback, Widget) 가이드 및 전용 **Console**과 같은 통합개발 환경을 제공 받을 수 있습니다.
