# DATA

뮤직 서비스(VIBE, Line Music)의 Data Driven Decision 과 추천 서비스를 위한 데이터를 수집 및 가공, 제공하는 업무입니다. 세부 업무는 다음과 같습니다.

\- 이용자/정산 로그 ETL&#x20;

\- 서비스 지표 시각화&#x20;

\- 추천 서비스 데이터 생성&#x20;

\- VIBE 차트 데이터 생성&#x20;

\- 서비스 품질 향상 (에러 탐지, 검색 성능 등)

* 기술 스택   &#x20;

&#x20;\- API 제공을 위한 서버: Java, Spring boot   &#x20;

&#x20;\- 데이터 처리/저장: Hadoop, Hive, Spark, Trino, Hbase, MySQL, Kafka   &#x20;
