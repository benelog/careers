# Music Platform

### IF Grow Together THEN GOTO 'Music Platform'

뮤직 플랫폼 서버 개발팀은 바이브 & 라인 뮤직에서 함께 서비스를 될 수 있는 공통 플랫폼 서버를 개발하고자 노력하고 있으며 네이버(한국) / 라인(일본)에 함께 서비스 되는 공통 서비스 서버를 개발하는 만큼 글로벌 서비스를 위한 표준화된 기술 환경과 대용량 요청들을 빠르게 처리 할 수 있는 기술 스택에 대해 고민하고 있습니다.

### 개발환경

* 기술스택 : Kotlin, Java, Spring Cloud, QueryDSL, Gradle, Junit5, K8S, MongoDB, MySQL, KLint
* 업무도구 : Github Ent, Confluence, Slack, Works
* 협업도구 : Github Ent MR 코드리뷰 및 Intellij Code with Me를 이용한 원격 코드 리뷰
* CI/CD : Github Ent MR Webhook CI / K8S 기반의 사내 배포 플랫폼 시스템

### 자격요건

* 관련 분야에서 1년 이상 실무 경험 보유

### 우대사항

* MSA 기반의 애플리케이션 구현 경험
* Kotlin/Spring 기반의 서버 개발 경험
* 모바일 앱/웹 서비스 환경의 서버 API 개발 경험
* K8S와 같은 컨테이너 환경에 대한 개발 경험
* MongoDB 개발 경험을 보유하신 분
